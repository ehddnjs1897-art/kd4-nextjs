## 1. 톤 프로파일

### 1.1 톤 축(Tone Axes)

| 축 | 포지션 | 근거 |
|---|---|---|
| formal ↔ informal | **세미-포멀** (정중한 존댓말 + 가벼운 구어체 혼용) | `"우리는 양산형 배우를 찍어내는 공장식 학원을 거부합니다."` (L545), `"부담 없이 문의하세요."` (L1283) |
| warm ↔ sharp | **warm-sharp 혼합** (선언문은 sharp, 안내문은 warm) | sharp: `"양산형 배우를 찍어내는 공장식 학원을 거부합니다"` (L545) / warm: `"어떤 클래스가 맞는지 모르겠다면 — 부담 없이 문의하세요."` (L1283) |
| aspirational ↔ grounded | **grounded-aspirational** (숫자·경력으로 지지된 포부) | `"프로 배우 400명+ 액팅 코칭"` (L685), `"300+ 배우 코칭 / 3년+ / 70명+"` (L600-602) + `"배우지망생 → 진짜 배우"` (L820-822, L1321) |
| 브랜드 자세 | **선언형 + 동료형** | "우리는 ~ 거부합니다" / "우리는 ~ 성장시키는" (L545-548) 주어 선택, 배우를 "동료"로 바라봄 |
| 과잉/절제 | **절제된 감정 강조** — 형용사 남발 없이 동사·명사로 주장 | `"연기하지 않는 연기"` (L494) — 4글자만으로 철학 전달 |

### 1.2 문체 특징

- **짧은 문장 선호**. 한 문장 = 한 주장. (L545, L548, L1311)
- **대비 구조(X가 아닌 Y)**: `"양산형 배우를 찍어내는 공장식 학원을 거부 → 배우를 성장시키는 KD4"` (L545-548)
- **"→" 기호를 이행/여정 메타포로 사용**: `"배우지망생 → 진짜 배우"` (L820-822, L1321), `"CASTING → /join 핸드오프"` 섹션명, CTA 말미 `→` 일관 (L191, L570, L1038, L1194, L1347)

### 6.2 CTA 패턴 규칙 (관찰된 규범)

- **주 CTA (navy 배경)**: 한국어 + **반각 space + "→"** 꼬리. (예: `"무료 상담 신청 →"`)
- **보조 CTA / 카카오 (ghost)**: 화살표 없음, 명사형 (예: `"카카오 상담"`, `"카카오로 문의하기"`, `"배우 갤러리"`)
- **클래스 카드 CTA**: `gold` 배경 + `"상담 신청 →"` (ClassCard 공통, L172-192)
- **Hero intro의 1차 CTA는 "클래스 알아보기"** — 상담이 아닌 **탐색 유도**가 먼저 옴.

## 8. 내부 모순 / 불일치

### 8.1 숫자 불일치

- **"300+" (Stats L600) vs "400명+" (Director L685)**: 같은 "코칭 배우 수"로 읽힐 여지. Stats는 스튜디오 집계, 400+는 대표 개인 커리어로 구분되지만 카피에 **구분 라벨이 약함**.
- **"70명+" (현재 수강배우, L602)** — "수강배우"라는 합성어 사용. CLAUDE.md 규칙("수강생 금지 → 동료 배우/KD4 배우")과의 **경계에 걸친 표현**. 엄격히 보면 "수강" 접두가 남아 있음.

### 8.2 CTA 문구 4종 혼재

동일한 **"상담 신청"** 기능을 두고 4종 카피가 공존:

| 카피 | 라인 | 맥락 |
|---|---|---|
| 무료 상담 신청 (→) | L1273, L1347 | Contact h2 · 최종 CTA |
| 수강 상담 신청 → | L1194 | Classes 섹션 CTA |
| 상담 신청 → | L191 | ClassCard 개별 CTA |
| 카카오 상담 / 카카오로 문의하기 | L1198 / L1365 | 카카오 보조 CTA (명사 다름) |

→ **표준화 후보**: `무료 상담 신청 →` (가장 명확)

### 8.3 섹션 제목 영문/한글 혼재 기준 불일치

- About H2: `배우지망생 → 진짜 배우` (한글 중심)
- Classes H2: `CLASSES` (영문 단독, L1134)
- How It Works H2: `How It Works` (영문 + 공백 형식, L1053)

→ 영문 H2(`CLASSES`, `How It Works`) 2개만 영문. 대부분 한글. 기준 모호.

### 8.4 "How It Works" 내부 카피와 전체 "About 3단계" 개념 중복

- **About L838-859** 의 STEP 01/02/03 (메소드 → 포트폴리오 → 캐스팅)
- **How It Works L1067-1081** 의 01/02/03 (상담/수강신청 → 첫수업/피드백 → 성장/캐스팅)

## 9. 톤 보존을 위한 카피 작성 규칙 (요약)

1. **선언문은 "우리는 ~" 주어로.** 부정-긍정 대비 구조 유지.
2. **CTA 주 버튼은 한국어 + " →"**. 보조/카카오는 화살표 금지.
3. **변환 약속은 "X → Y" 포맷**. ("배우지망생 → 진짜 배우"가 공식)
4. **숫자는 반드시 `+` 꼬리**. (300+, 400명+, 70명+, 2000만뷰+)
5. **"수강생" 금지**. "배우", "동료 배우", "KD4 배우", "멤버" 사용.
6. **베이직/입문 맥락엔 "연기 경험 없어도 OK" 필수**.
7. **Eyebrow는 대문자 영문**(spaced letter-spacing). `CURRICULUM`, `PROCESS`, `CONTACT` 등.
8. **클래스 카드 quote는 1줄 시적 문구**(명사형 + 감정어).
9. **철학 키워드 세트 고정**: 마이즈너 테크닉 / 이바나 처벅 테크닉 / 감정 해방 / 충동·본능 회복 / 연기하지 않는 연기 / 소수정예.
10. **브랜드 이름은 "KD4 액팅 스튜디오"** (풀), "KD4"(약어), "유익액터스"(법인). 혼용 금지.
