# 보험 가이드 (KD4 운영 위험 헤지)

> **용도**: KD4 같은 액팅 스튜디오·매니지먼트가 가입해야 할 보험 정리
> **원칙**: 보험은 **비용**이 아니라 **리스크 이전**. 단 1건 사고로 회사가 흔들리지 않게.

---

## 0. KD4의 주요 위험 (Risk Map)

| 위험 카테고리 | 발생 빈도 | 손해 규모 | 권장 보험 |
|---|---|---|---|
| 수업 중 멤버 부상 | 중 | 중~상 | 영업배상책임보험 |
| 화재·시설 사고 | 저 | 상 | 시설소유자(관리자)배상책임 |
| 강사의 업무상 실수 | 저 | 중 | 전문직배상책임 (Optional) |
| 행사·발표회 사고 | 저 | 중 | 이벤트 일일 보험 |
| 직원 산재 | 중 | 중 | **산업재해보상보험 (법정 의무)** |
| 개인정보 유출 | 저 | 상 | 사이버 책임 보험 |
| 미성년 안전 사고 | 저 | 상 | 영업배상 + 어린이 사고 특약 |
| 화재로 인한 영업중단 | 저 | 상 | 영업휴지손해 보험 |

---

## 1. 영업배상책임보험 (Comprehensive General Liability)

### 1-1. 가입 사유 ★★★★★ (가장 우선)

**보장 내용**
- 사업 운영 중 제3자(멤버·방문객)의 **신체 사고**·**재물 손해**
- 합의금·치료비·소송비용·변호사 보수

**KD4 시나리오**
- 멤버가 수업 중 무리한 동작으로 발목 부상
- 발표회 중 무대 장비 낙하로 멤버 부상
- 학원 시설(거울·소품)에 의한 사고

### 1-2. 가입 시 체크포인트

| 항목 | 권장 |
|---|---|
| 보장 한도 (1사고 / 연간 누적) | 1억 / 5억 이상 |
| 자기부담금 (deductible) | 30만~50만원 (낮을수록 보험료↑) |
| 면책 사유 | 고의·중과실·제외 활동 종목 확인 |
| 보험료 | 연 30만~100만원 (학원 규모 따라) |

### 1-3. 특약 검토

- **시설소유자배상** (시설 자체 위험)
- **어린이/청소년 사고 특약** (미성년 멤버 다수)
- **공연·이벤트 일일 보험** (발표회·외부 행사)
- **재산종합** (영업 시설 자체 화재·도난)

### 1-4. 추천 보험사

- 삼성화재·DB손해보험·KB손해보험·현대해상 등 모두 영업배상 상품 보유
- **3사 이상 견적 비교** 권장

## 2. 산업재해보상보험 (의무)

### 2-1. 법정 의무

- **근로기준법상 근로자가 1명 이상**이면 의무 가입
- **사용자 100% 부담**, 근로자 공제 없음
- 사업 개시 14일 이내 신고

### 2-2. 가입 절차

```
1. 근로복지공단 www.kcomwel.or.kr 접속
2. 사업장 정보 입력 → 보험관계 성립신고
3. 매년 3월 31일까지 보수총액신고서 제출
```

### 2-3. 보험료 산정

```
연간 임금총액 × 업종별 요율
- 학원·교육업 평균 요율: 약 0.7~0.9%
- 정확한 요율은 공단 홈페이지 또는 노무사 문의
```

### 2-4. 미가입 시 불이익

- 산재 발생 시 근로자 보험급여는 정상 지급
- 사용자에게 **미가입 가산금** + 보험료 소급
- 별도 형사처벌 가능
- 평판 손상

## 3. 전문직배상책임보험 (Professional Liability)

### 3-1. 검토 대상 (Optional)

- 강사의 잘못된 코칭·지도로 멤버가 입은 손해
- 매니저의 부적절한 캐스팅 매칭으로 멤버가 입은 손해
- 잘못된 법률·세무 자문 (해당 외주 자문사 보험)

### 3-2. KD4 자체 가입 필요성

- 일반 학원·매니지먼트는 **영업배상에 포함되는 경우 多**
- 전속 멤버 다수 + 대형 광고 캐스팅 자주 진행 시 별도 고려

---

## 4. 사이버 책임 보험 (Cyber Liability)

### 4-1. 검토 사유

- **개인정보 유출** 시 법적 책임 (개인정보보호법 과징금·손해배상)
- 랜섬웨어·해킹 복구 비용
- 영업 중단으로 인한 손해

### 4-2. KD4 위험

- 멤버 50명+ 개인정보 보관
- 결제 정보·실연 영상·연락처
- Supabase·Vercel 인프라 의존

### 4-3. 가입 검토 기준

- 멤버 100명 이상 + 개인정보·결제 다수 처리 → 적극 검토
- 보험료 연 50만~200만원 (보장 한도 따라)

---

## 5. 미성년 멤버 안전 사고 대비

### 5-1. 영업배상에 어린이 특약 추가

- 미성년 멤버 다수 → 어린이/청소년 사고 특약 필수
- 보장 한도 상향 (1억 → 3억 이상)

### 5-2. 사고 발생 시 대응 절차 (보험 외 추가)

```
1. 사고 즉시 응급조치 + 119
2. 보호자 즉시 연락
3. 보험사 사고 통지 (24시간 이내)
4. 사고 경위 기록 (시각·장소·증인·CCTV 확보)
5. 변호사 자문 (중대 사고 시)
6. 합의·소송 대응
```

## 6. 화재·재산 보험

### 6-1. 검토 기준

- 학원 시설·장비·교재의 자산 가치
- 임차료·재무 손실 대비 영업휴지손해 보험

### 6-2. 가입 패키지 예시

| 보장 | 한도 |
|---|---|
| 시설·집기 화재·도난 | 1억 |
| 영업휴지손해 (월) | 500만 × 6개월 |
| 임대인 배상 (시설 사고) | 1억 |

→ 연 30만~80만원

---

## 7. 임원배상책임보험 (D&O Insurance)

### 7-1. 검토 대상

- KD4가 법인이고 대표이사·이사 다수
- 주주·멤버·외부 분쟁으로 임원 개인 책임 발생 가능 시

### 7-2. 신생 법인 권장도

- 1인 법인·소규모는 통상 미가입
- 외부 투자 유치 후·이사회 확대 시 적극 검토

---

## 8. 보험사 선택 시 체크리스트

```
□ 보장 한도가 KD4 위험 규모에 적정한가
□ 자기부담금이 운영 가능한 수준인가
□ 면책 사유 확인 (고의·중과실 외에 제외 항목)
□ 갱신 시 조건 변경 가능성
□ 사고 처리 속도·평가 (보험소비자보호원 평가)
□ 대리점·설계사 신뢰성
□ 보험금 청구 절차 명확성
□ 변호사 선임권 (보험사 지정 vs KD4 선임 가능)
□ 면책 조항에 "허위 진술 시 보상 거부" 등 포함 여부
```

## 9. 보험 외 위험 헤지

### 9-1. 약관·계약서

- 환불·해지 조항 명확화 (분쟁 예방)
- 면책 조항 (불가항력·자기 책임)
- 손해배상 한도 명시

### 9-2. 운영 매뉴얼

- 안전 수칙 (수업 중 부상 예방)
- 미성년 응대 매뉴얼
- 시설 점검 일지

### 9-3. 기록 관리

- CCTV (사고 입증)
- 출석부·동의서·서명본 보관
- 카톡·이메일 백업

## 10. KD4 우선 순위 가입 권장 (단계별)

```
Phase 1 (즉시) ─ 법정 의무
  └── 산업재해보상보험 (직원 1명 이상 시)

Phase 2 (1개월 이내) ─ 핵심 영업 위험
  └── 영업배상책임보험 (어린이 특약 포함)
  └── 시설소유자배상

Phase 3 (분기 내) ─ 재산·중단
  └── 화재·시설 재산보험
  └── 영업휴지손해 보험

Phase 4 (반기 내) ─ 정보·전문 위험
  └── 사이버 책임 (멤버 100명+ 시)
  └── 전문직배상 (대형 매니지먼트 거래 시)

Phase 5 (필요 시) ─ 임원·투자
  └── 임원배상책임 (외부 투자 후)
```

## 11. 연간 보험 비용 예상치 (참고)

| Phase | 항목 | 연 보험료 |
|---|---|---|
| Phase 1 | 산재 | 임금 × 0.7% |
| Phase 2 | 영업배상 (1억/5억) | 30만~50만원 |
| Phase 2 | 시설소유자배상 | 20만~30만원 |
| Phase 3 | 재산보험 (1억) | 30만~50만원 |
| Phase 3 | 영업휴지 | 20만~30만원 |
| Phase 4 | 사이버 (1억) | 50만~100만원 |
| **합계** | (Phase 1~3 패키지) | **연 100만~200만원** |

→ KD4 같은 학원이 화재·멤버 부상으로 천만 원~억 단위 손해 발생 시
   연 100~200만원 보험료 vs 한방에 위험 종결 — 비용 대비 절대 우위

## 12. 사고 발생 시 통지 절차

```
사고 발생 (예: 멤버 수업 중 부상)
  ↓ 즉시 응급조치 + 119
  ↓
24시간 이내 ─→ 보험사 사고 신고 (전화 또는 앱)
  ↓
사고 보고서 작성 (서면)
  - 일시·장소·경위·당사자·증인·치료병원
  ↓
보험사 사고 조사 (사정인 배치)
  ↓
손해 평가 → 보험금 산정
  ↓
합의 (멤버/보호자) → 보험금 지급
```

**주의**
- 사고 직후 SNS·언론 발언 절대 ❌ (보험사·법적 입장 약화)
- 멤버·보호자에게 "다 책임지겠다"는 약속 ❌ (보험사 면책 사유)
- 모든 소통은 기록 보존
