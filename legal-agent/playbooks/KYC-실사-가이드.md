# 거래 상대방 KYC·실사 가이드 (사기·도산 방지)

> **용도**: 출연료·외주비·라이선스 계약 등 **결제·정산이 발생하는 거래**의 상대방 진위 확인
> **목적**: ① 페이퍼컴퍼니·사기 회피 ② 도산 위험 사전 감지 ③ 분쟁 시 책임 주체 명확화
> **시점**: 계약 체결 전 (반드시) + 분기별 (전속·장기 거래)

---

## 0. 빨간 신호 체크 (1분 안에 끝)

```
□ 회사명만 알려주고 사업자등록번호·주소·대표자 누락
□ 회사 홈페이지·SNS·언론 검색 시 정보 거의 없음
□ 회사 이메일 도메인이 무료 도메인(@naver, @gmail) 사용
□ 대표자 직접 만나기 거부, 중간 매니저만 응대
□ 사업자등록증 사본 요청 시 거부·지연
□ "급한 일정"으로 계약 검토 시간 안 주려고 함
□ 계약금만 먼저 입금 요구 (결과물·근거 없이)
□ 상대방이 본인 명의 계좌가 아닌 제3자 계좌로 결제 요구
□ 회사 등기상호·사업자등록증·통상 호칭이 모두 다름
□ 대표자 인적 사항 검색 시 과거 분쟁·기사 다수
```

→ 위 중 3개 이상 ✅ **계약 보류 + 추가 실사**

## 1. 개인사업자 KYC (1차)

### 1-1. 받아야 할 서류

| # | 서류 | 발급처 | 비고 |
|---|---|---|---|
| 1 | 사업자등록증 사본 | 본인 발급 (홈택스) | 사업장 주소·대표자 확인 |
| 2 | 신분증 사본 | 본인 | 대표자 본인 확인 |
| 3 | 통장 사본 | 본인 | 대표자 명의 일치 확인 |
| 4 | 사업자등록증명원 (3개월 이내) | 홈택스 무료 발급 | 사업자 상태(휴·폐업 여부) |

### 1-2. 진위 확인 방법

**무료**
- **국세청 사업자등록 상태 조회** (홈택스)
  - https://www.hometax.go.kr → 사업자등록상태조회
  - 사업자등록번호 입력 → 활동·휴업·폐업 여부 즉시 확인
- **사업자 진위 확인** (납세자 동의 시)
  - 위 페이지에서 사업자 진위 확인 (성명·주민번호 일부 매칭)

**유료**
- **NICE·KCB 개인사업자 신용평가** (10,000~30,000원/건)

## 2. 법인 KYC (1차)

### 2-1. 받아야 할 서류

| # | 서류 | 발급처 | 비고 |
|---|---|---|---|
| 1 | 사업자등록증 사본 | 법인 본인 | |
| 2 | 법인등기부등본 (3개월 이내) | 대법원 인터넷등기소 | **반드시 발급 후 확인** |
| 3 | 법인인감증명서 | 등기소 | 계약 체결자 권한 확인용 |
| 4 | 법인 통장 사본 | 법인 본인 | 송금 계좌 일치 확인 |
| 5 | 사용인감계 (대표가 직접 서명 안 할 때) | 법인 본인 | 대리인 권한 확인 |

### 2-2. 법인등기부등본에서 봐야 할 핵심

```
1. 상호
   - 계약서 표지·본문에 적힌 회사명과 일치하는가?
   - "줄임말·통상 호칭"이 등기상호와 다른가?

2. 본점 소재지
   - 계약서 주소와 일치?
   - 페이퍼컴퍼니 의심 시 실제 주소 방문 검증

3. 대표이사·이사진
   - 서명자가 등기상 대표인가? (퇴임한 사람이 서명한 사례 多)
   - 공동대표인가? (공동대표는 1명 서명만으로 무효)

4. 자본금
   - 자본금 100만원·1,000만원짜리 페이퍼컴퍼니 확인
   - 계약 금액이 자본금의 10배 이상이면 도산 시 회수 불가

5. 설립일자
   - 1년 미만 신생 법인 → 추가 검증 필요
   - 사업 양수 사례 vs 신설 사례

6. 회사 변경 이력
   - 상호 변경·대표 변경·소재지 이전 빈도
   - 변경 잦으면 의심
```

### 2-3. 추가 검증 (유료)

| 서비스 | 비용 | 내용 |
|---|---|---|
| 신용평가서 (NICE·KIS·KCB) | 5만~30만원 | 신용등급·재무·도산 위험 |
| 결산공고 (전자공시 DART) | 무료 (외감 법인만) | 매출·자산·부채 |
| 사업자 신용정보 | 무료 (정부 사이트 일부) | 휴·폐업 정보 |

→ **계약 금액 1,000만원 이상**이면 신용평가서 발급 권장 (약 5만원)

## 3. 검증 후 확인할 일관성

### 3-1. "삼중 일치" 체크

다음 3가지가 **모두 일치**해야 정상:

```
계약서 표지 회사명 = 계약서 본문 회사명 = 법인등기 상호
계약서 대표자 성명 = 등기 대표자 성명 = 법인인감 명의
입금 받을 계좌 명의 = 법인명 또는 등기 대표자명
```

**실제 KD4 케이스 (불일치 예시)**
- 표지: "세이온미디어㈜"
- 본문: "몬스터유니온"
- 사업자번호: 106-86-89059 (같은 대표이사 임성균)

→ 같은 대표라도 **법인은 별개의 법인격**. 어느 쪽이 책임지는지 명시 필요.

### 3-2. 대처법 (3중 불일치 시)

**옵션 A**: 명의 통일 요청
- 한쪽 회사명으로 통일
- 모든 조항·서명·결제 계좌 동일하게 맞춤

**옵션 B**: 연대 보증 추가
- "갑(세이온미디어㈜)과 갑의 관계 법인(몬스터유니온)은 본 계약상 의무를 연대하여 부담한다"
- 양 법인 모두 서명·날인 + 양 법인 모두 사업자등록증·등기부 제출

## 4. 계약자 권한 검증 (서명자가 진짜 권한자인가)

### 4-1. 법인 서명 패턴

| 서명자 | 검증 서류 |
|---|---|
| **대표이사 직접** | 법인등기상 현재 대표 확인 + 법인인감 |
| **사용인 / 이사** | 사용인감계 + 사용인감카드 + 위임장 |
| **외부 대리인** | 위임장 + 위임받은 자의 신분증 |

### 4-2. 자주 보는 사기 패턴

1. **퇴임한 대표가 서명** — 회사 인감만 받고 등기 미확인 시 사고
2. **공동대표 1명만 서명** — 공동대표는 양측 모두 서명 필요
3. **위임장 위조** — 위임장에 위임인 인감 + 인감증명서 받아 확인
4. **본사 vs 지점/계열사** — 계약 책임 주체가 본사인지 지점인지 명확화

## 5. 결제·송금 KYC

### 5-1. 받는 입장 (KD4가 출연료·외주비 받을 때)

- 상대방 명의 계좌인지 확인 (제3자 계좌 ❌)
- 송금자 명·일자·금액 일치 확인
- **세금계산서·현금영수증** 정확히 발급/수령

### 5-2. 지급 입장 (KD4가 외주비·강사료 지급 시)

- 받는 분 명의 계좌 확인 (개인사업자/법인 명의)
- 사업소득 3.3% 원천징수 → 다음달 10일 지방세·소득세 신고
- 세금계산서 발급 거래 시 적격 증빙 보관 (5년)

### 5-3. 송금 사기 회피

- 상대방이 "급하게 계좌 바꿨다"고 다른 번호 알려주면 ⚠️
- 반드시 **이메일이 아닌 직접 통화**로 계좌 변경 확인
- 큰 금액은 분할 송금 + 1차 송금 후 도착 확인 후 2차

## 6. KD4 입장에서 추가 체크 (출연·매니지먼트)

### 6-1. 외부 제작사·캐스팅 디렉터

- 제작 이력 (IMDB·KMDB·언론기사)
- 대표 이력 (LinkedIn·공식 프로필)
- 동종 업계 평판 (다른 매니지먼트사·배우 추천 여부)
- 회사 SNS·홈페이지 운영 상태
- 사무실 실제 운영 여부 (방문 가능 시 직접 확인)

### 6-2. 광고주 / 모델 의뢰

- 광고주가 직접 의뢰인지 vs 광고대행사 경유인지
- **광고대행사 경유 시**: 광고대행사 KYC + 광고주 KYC 둘 다
- 광고 컨셉·제품 정합성 (위법·부도덕 제품 거절)
- 광고비 정산 주체 명확화

### 6-3. 외주 강사·코치 영입

- 학력·경력 진위 확인 (졸업증명서·재직증명서)
- 자격증 진위 (한국연극협회·관련 협회)
- 형사 기록 (필요 시 신원조회 동의)
- 미성년 멤버 응대 가능 인성·이력

## 7. 계약 후 모니터링 (전속·장기 거래)

### 분기별 체크
- [ ] 사업자등록 상태 (홈택스 재확인)
- [ ] 법인등기부 변경 사항 (대표 교체·자본금 감자 등)
- [ ] 최근 언론 보도 (분쟁·사고)
- [ ] 정산·결제 정시 이행 여부

### 위험 신호
- 정산 지연 (60일 이상) → 도산 의심
- 대표 교체 → 신규 대표와 계약 재확인 필요
- 자본금 감자 → 회사 규모 축소 → 채권 회수 위험
- 사무실 이전 (주소 불분명) → 잠적 가능성

## 8. KD4 자체 체크리스트

```
[1차 — 받자마자]
□ 사업자등록증 사본
□ 법인등기부등본 (법인일 시)
□ 신분증 사본
□ 통장 사본

[2차 — 1시간 안에]
□ 홈택스 사업자 상태 확인
□ 등기상호 vs 계약서 회사명 일치 확인
□ 대표자 vs 서명자 일치 확인
□ 입금계좌 명의 일치 확인

[3차 — 1일 안에]
□ 회사·대표 인터넷 검색 (분쟁·기사)
□ 사업장 주소 지도 검색 (페이퍼컴퍼니 의심)
□ 회사 SNS·홈페이지 운영 상태
□ (필요 시) 신용평가서 발급

[결정 — 모두 정상 시 계약 진행]
□ 계약서 회사명·서명자·계좌 일치
□ 사업자등록증 사본 보관
□ 등기부등본 사본 보관
```
