# 분쟁 대응 가이드 (KD4 위기관리)

> **목표**: 분쟁이 발생했을 때 **확전 방지 + 증거 보존 + 합리적 종결**
> **원칙**: ① 침착 ② 기록 ③ 검토 ④ 응대

---

## 0. 분쟁 발생 첫 24시간 — 무조건 해야 할 일

### ✅ 즉시 (1시간 이내)
- [ ] 사실관계 시간순 정리 (메모장에 기록 — 발생 일시·관련자·상황)
- [ ] 관련 카톡·이메일·SNS 캡처 (도메인·URL·시각 포함)
- [ ] 관련 계약서·결제 기록·CCTV 영상 위치 확인
- [ ] **단독 응대 금지** — 상대방에 충동적 답신 ❌

### ✅ 6시간 이내
- [ ] 회사 내부 1차 보고 (대표·법무 담당)
- [ ] 영업배상책임보험 가입 여부 확인 → 가입 시 보험사 사고 통지
- [ ] 변호사·노무사·세무사 연락처 정리 (자문 의사 확인)

### ✅ 24시간 이내
- [ ] 사실관계 + 증거 정리한 1쪽짜리 요약 작성
- [ ] 분쟁 카테고리 분류 (아래 1~6번 참조)
- [ ] 대응 라인 결정: 자체 응대 / 변호사 위임 / 보험사 위임
- [ ] 상대방에 1차 응대 (수동적·중립적 — "확인 후 회신드리겠다")

## 1. 멤버 환불 분쟁

### 시나리오: "수강 중인데 환불해주세요"

```
1단계 ─ 약관·결제 정보 확인
  ├─ 결제일·결제수단·금액
  ├─ 개강일·총 수강시간·현재 진행 회차
  └─ 학원법 시행령 별표 4 환불 비율 계산

2단계 ─ 환불 비율 결정
  ├─ 회사 귀책 (강사 변경·휴강 등) → 100% 환불
  ├─ 멤버 사유 + 환불 가능 구간 → 약관 기준
  └─ 멤버 사유 + 환불 불가 구간 → 거부 가능
      ※ 보강·할인·이월 등 대안 제시 (선의)

3단계 ─ 응대
  ├─ 환불 가능 → 카톡 템플릿 B1
  ├─ 환불 거부 → 카톡 템플릿 B2
  └─ 미동의 → 내용증명 양식 2

4단계 ─ 확전 시
  ├─ 한국소비자원 분쟁조정 신청 (멤버) → 회사 협조 + 정당 사유 소명
  ├─ 학원감독청(교육청) 민원 → 신속 응대 + 사실관계 입증
  └─ 민사 소액심판 (3,000만원 이하) → 변호사 자문
```

### 위험 신호 — 즉시 변호사 자문
- 환불 요구 + **소비자원 또는 교육청 신고** 협박
- 환불 요구 + **SNS 폭로** 협박
- 환불 요구 + **언론 제보** 협박
- 학원법 위반 가능성 (실제로 위반한 케이스인 경우)

## 2. 멤버 SNS 비방·평판 분쟁

### 시나리오: "전 멤버가 인스타에 KD4 욕을 올림"

```
1단계 ─ 사실관계 확인
  ├─ 내용이 사실인가 vs 허위·왜곡인가
  ├─ 일부 사실 + 일부 허위 → 신중 판단 필요
  └─ 의견 표명인가 사실 적시인가

2단계 ─ 증거 보존
  ├─ 게시물 URL·캡처·게시 시각·작성자 식별 정보
  ├─ 공증사무소 통한 공정증서 작성 가능 (증거력 강화)
  └─ 인터넷주소 보존서비스 (한국인터넷진흥원) 활용

3단계 ─ 대응 결정
  ├─ 일부라도 사실 → 내부 개선 우선. 법적 대응 비추천
  ├─ 허위·왜곡 → 다음 단계 진행
  └─ 의견 표명 → 명예훼손 성립 어려움 (수인한도)

4단계 ─ 단계별 대응
  ├─ a. 정중한 정정·삭제 요청 (카톡 D2)
  ├─ b. 내용증명 발송
  ├─ c. 플랫폼 임시조치 신청 (해당 SNS 신고)
  ├─ d. 명예훼손·정보통신망법 형사 고소
  └─ e. 손해배상·업무방해 민사 소송
```

### 명예훼손 성립 요건 체크
- [ ] **공연성** — 불특정 또는 다수가 인식 가능 (SNS 공개 게시 ✅)
- [ ] **사실의 적시** (의견 표명만으로는 미성립)
- [ ] **사회적 평가 저하 가능성**
- [ ] **특정성** — 회사·개인이 누구인지 식별 가능

### 형사 vs 민사 트레이드오프
| 항목 | 형사 | 민사 |
|---|---|---|
| 비용 | 무료 (고소장) | 인지대·송달료 |
| 효력 | 처벌 + 위협 효과 | 손해배상금 |
| 시간 | 6개월~2년 | 1~2년 |
| 리스크 | 무고죄 역공 (입증 약하면) | 패소 시 소송비용 부담 |
| 합의 | 처벌 면제 거래 가능 | 강제집행 가능 |

## 3. 출연료·정산 미지급 분쟁

### 시나리오: "촬영 끝났는데 출연료 안 줘"

```
1단계 ─ 계약·증거 정리
  ├─ 출연계약서 사본
  ├─ 촬영 출석부·콜시트·서명한 회차 명부
  ├─ 정산 요청 카톡·이메일 기록
  └─ 미지급 금액·지연 기간 산정

2단계 ─ 단계별 독촉
  ├─ a. 카톡 1차 (C1) — 정중한 확인
  ├─ b. 카톡 2차 (C2) — 60일 경과 시 강한 어조
  ├─ c. 카톡 3차 (C3) — 내용증명 발송 사전 통지
  ├─ d. 내용증명 발송 (양식 1)
  └─ e. 회신 없거나 거절 시 다음 단계

3단계 ─ 본격 법적 조치
  ├─ 3,000만원 이하 → 소액심판 (변호사 없이도 가능)
  ├─ 3,000만원 초과 → 일반 민사
  ├─ 지급명령 신청 (간이·신속·증거 약하면 본안 회부)
  └─ 가압류 (제작사 자산 동결)

4단계 ─ 형사 검토
  ├─ 단순 미지급 → 형사 미성립 (민사)
  ├─ 처음부터 지급 의사 없이 출연 유도 → 사기 가능
  └─ 임금체불 (근로자성 인정 시) → 노동청 진정
```

### 지연이자
- 민사 법정이자 연 5% (민법 제397조)
- 상사 법정이자 연 6% (상법 제54조 — 거래 양 당사자가 상인일 때)
- 소송촉진법 연 12% (소제기 후 판결 확정 시까지)

## 4. 강사·전 직원 영업비밀·경업 분쟁

### 시나리오: "퇴사한 강사가 우리 커리큘럼 들고 학원 차림"

```
1단계 ─ 영업비밀성 입증 준비
  ├─ 비공개성 (외부에 공개되지 않음)
  ├─ 경제적 유용성 (사업상 가치)
  ├─ 비밀관리성 (보안조치 — 비밀유지서약서·접근 제한·문서 표시)
  └─ 위 3요건 모두 충족해야 영업비밀

2단계 ─ 증거 확보
  ├─ 비밀유지서약서 사본
  ├─ 경업금지 약정 사본
  ├─ 영업비밀 표시 자료 (커리큘럼 PDF·교재 등)
  ├─ 침해자가 모방한 자료 (홈페이지·교재·SNS)
  └─ 멤버 이전 정황 (동일 시기 멤버 단체 이동)

3단계 ─ 대응 단계
  ├─ a. 내용증명 (양식 4)
  ├─ b. 경업금지 가처분 신청 (긴급한 경우)
  ├─ c. 영업비밀 침해 가처분 (사용 중단·반환)
  └─ d. 본안 — 부정경쟁방지법 제10조 손해배상

4단계 ─ 형사 검토
  ├─ 부정경쟁방지법 제18조 — 5년 이하 징역
  ├─ 업무상배임 (재직 중 빼돌린 경우)
  └─ 정보통신망법 (데이터 무단 반출 시)
```

### 경업금지 약정의 한계
- 기간 1년·동일 권역 정도가 합리적
- 너무 광범위하면 **무효** (직업선택의 자유 침해)
- 대가성 보상 없으면 **무효 가능성 ↑**
- 단순 강사 → 영업비밀 접근 약하면 경업금지 적용 좁게 해석

## 5. 미성년 멤버·보호자 분쟁

### 시나리오: "부모가 손해배상 청구"

> **⚠️ 미성년 분쟁은 사회적 비난 + 언론 보도 위험 ★★★**
> **단독 응대 절대 금지. 즉시 변호사 자문.**

```
1단계 ─ 즉시 조치 (1시간 이내)
  ├─ 멤버 안전 확보 (성희롱·체벌·학대 의혹은 별도)
  ├─ 대표 + 변호사 + 보험사 즉시 통지
  └─ 모든 외부 응대 금지 + 1인 대응 창구 단일화

2단계 ─ 사실관계 기록 (24시간 이내)
  ├─ 수업 일지·CCTV·증인 진술서
  ├─ 멤버·강사·동료 진술 (각자 별도)
  └─ 사고 발생 추정 시점 ±2주 일정

3단계 ─ 응대 라인 결정
  ├─ 회사 단독 응대 (경미·명확) → 카톡 F2
  ├─ 변호사 위임 응대 (중대·복잡)
  └─ 보험사 위임 응대 (상해·재산 손해)

4단계 ─ 합의 vs 소송
  ├─ 사실관계 명확 + 회사 책임 → 신속 합의 권장 (확전 회피)
  ├─ 사실관계 다툼 + 회사 책임 없음 → 변호사 통해 정식 응대
  └─ 형사 고소 동반 → 절대 단독 응대 ❌
```

### 미성년 관련 보고 의무
- **아동학대 의심** — 아동학대처벌법상 신고 의무 (24시간)
- **성희롱·성폭력 의심** — 성폭력방지법상 신고 의무
- **자살·자해 위험** — 보호자 즉시 통보 + 정신건강복지센터 연계

## 6. 외주·외부 사업자 분쟁

### 시나리오: "외주 영상 편집업체가 결과물 안 줌"

```
1단계 ─ 계약·증거 확인
  ├─ 외주용역계약서
  ├─ 착수금 송금 영수증
  ├─ 진행 상황 메시지·이메일
  └─ 납품 기한 vs 실제 경과 일수

2단계 ─ 정중 독촉 (카톡)
3단계 ─ 내용증명 (양식 5 — 계약 해지 + 손해배상 검토)
4단계 ─ 조치
  ├─ 착수금 반환 청구 (해지 후)
  ├─ 대체 업체 발주 차액 손해 청구
  └─ 영업방해·계약불이행 손해 청구
```

### 외주 분쟁에서 자주 놓치는 것
- **raw 소스·작업 파일** 확보 — 미납품 시 대체 작업 가능
- **계약상 기한 명확화** — "최대한 빨리" 같은 모호한 표현 ❌
- **단계별 검수 + 단계별 지급** — 위험 분산

## 7. 분쟁 대응 시 절대 하지 말 것

| ❌ | 이유 |
|---|---|
| 충동적 카톡·SNS 응대 | 캡처되어 증거화 |
| 욕설·인신공격·비방 | 명예훼손·모욕 역공 |
| "법적 조치 할 거다" 협박조 | 협박죄·공갈죄 위험 |
| 증거 인멸 (메시지 삭제·CCTV 삭제) | 형사 처벌 + 불리 추정 |
| 멤버에게 합의 강요 | 강요죄·부당이득반환 |
| 변호사 자문 없이 합의서 사인 | 불리한 조항 견과 |
| 보험 가입자인데 보험사 미통지 | 보험금 지급 거절 가능 |
| 분쟁 사실 SNS 공개 | 명예훼손 + 영업방해 역공 |

## 8. 분쟁 종결 후 사후 관리

- [ ] 합의서 PDF·서명본 보관
- [ ] 분쟁 사례 내부 공유 (재발 방지)
- [ ] 약관·계약서 개선 (분쟁 원인 조항 수정)
- [ ] 매뉴얼 업데이트
- [ ] 보험·법무 자문 검토 일정 등록

## 분쟁 대응 비용 예상치

| 단계 | 비용 |
|---|---|
| 카톡·이메일 응대 | 0원 |
| 내용증명 | 5,000~10,000원 |
| 변호사 자문 (단발) | 30만~100만원 |
| 변호사 수임 (자문 계약) | 월 30만~100만원 |
| 소액심판 (3,000만 이하) | 인지대 1만원~ |
| 일반 민사 | 인지대 + 변호사 보수 (보통 청구액 5~10%) |
| 형사 고소 | 0원 (변호사 수임 시 별도) |
| 가처분 | 보증보험 + 인지대 + 변호사 |
| 한국소비자원 분쟁조정 | 무료 |
