# 카톡 메시지 템플릿 (계약·분쟁·환불)

> 자주 보내는 실무용 메시지. 상황에 맞게 [빈칸]만 채워서 사용.
> 카톡은 친근하면서도 **증거력**을 갖춰야 한다 → 금액·날짜·근거 명확히.

---

## A. 계약서 검토 요청·수정 제안

### A1. 계약서 받았을 때 1차 확인 (수정 요청)

```
안녕하세요, 보내주신 [계약명] 잘 받았습니다.

검토 결과 몇 가지 사항만 수정 부탁드리고 사인하겠습니다.

1. [조항/내용] — 이 부분이 [이유] 때문에 [수정안]으로 변경 부탁드립니다.
2. [조항/내용] — [수정안]
3. [빈칸/누락 부분] — [채워야 할 내용] 확인 부탁드립니다.

수정본 받는 대로 바로 사인 후 회신드리겠습니다.
감사합니다.
```

### A2. 출연료 회차·합계 명확화 요청 (실제 KD4 케이스)

```
안녕하세요, [작품명] 출연계약서 검토 잘 마쳤습니다.

한 가지만 확인 부탁드립니다 —

제3조 출연료 조항에 회차 부분이 비어 있어서 명확히 채워주시면 좋겠습니다.
회당 20만원 × 8회 = 합계 1,600,000원 (세전)
위 내용으로 ⑦ 칸 채워주시고 보내주시면 바로 서명해서 회신드리겠습니다.

감사합니다.
```

### A3. 제작사 명의 통일 요청 (실제 KD4 케이스)

```
계약서 표지에는 [세이온미디어㈜]로, 본문에는 [몬스터유니온]으로 표기되어 있어서
명의를 통일해주시면 좋겠습니다.

어느 쪽이 정식 계약 당사자인지 확인해주시고,
사업자등록번호·대표자 일치 여부도 한 번 더 짚어주시면 감사하겠습니다.
```

### A4. 책임 한도 협상 (수임료 1배 → 10배 요청)

```
계약서 잘 받았습니다.

제[N]조 책임제한 한도가 수임료 1배로 되어 있는데,
저희가 [규모/거래액]을 위탁드리는 상황을 고려하면
합리적인 범위에서 [수임료 10배] 또는 [연간 수임료] 정도로 상향 부탁드려도 될까요?

(통상 세무·법무 분야 수임료 1배 한도가 관행인 것은 알고 있습니다.
다만 고의·중과실은 한도 적용 제외임을 확인하고자 합니다.)

검토 후 회신 부탁드립니다.
```

### A5. 자동갱신 조항 단축 요청

```
계약서 잘 봤습니다.

제[N]조 자동갱신 조항의 해지 통지 기간이 90일로 되어 있는데,
60일 또는 30일로 조정 가능할까요?
연간 계약을 검토할 시점이 너무 빨라지면 운영상 어려움이 있어서요.

다른 부분은 모두 이상 없습니다. 감사합니다.
```

## B. 환불·청약철회

### B1. 멤버 환불 요청 응대 (학원법 환불 비율 안내)

```
[멤버 성함]님 안녕하세요. KD4입니다.

환불 요청 잘 확인했습니다.
저희 약관과 학원법 시행령 기준에 따라 환불 비율 안내드립니다.

▪ 결제일: [날짜]
▪ 개강일: [날짜]
▪ 현재 진행 회차: [N회차 / 총 M회차]
▪ 진행률: 약 [X]%

학원법 환불 기준 적용:
- 총 수강시간의 [1/3 / 1/2 / 1/2 이후] 경과 → 환불 비율 [2/3 / 1/2 / 0]
- 환불 가능 금액: [____원]

위 금액은 환불 요청일로부터 5영업일 이내 [입금 계좌]로 입금드립니다.
확인 부탁드립니다.
```

### B2. 환불 거부 (법정 환불 불가 구간)

```
[멤버 성함]님 안녕하세요.

환불 요청 검토 결과 안타깝게도 환불이 어려운 점 안내드립니다.

학원법 시행령 별표 4에 따라
1개월 이내 단기 클래스는 총 수강시간의 1/2 경과 후 환불이 어렵습니다.
저희도 같은 기준을 약관 제[N]조에 명시해두었습니다.

▪ 결제일: [날짜]
▪ 개강일: [날짜]
▪ 진행 회차: [N회차 / 총 M회차] — 1/2 초과

대신 [잔여 회차 보강 수업 / 다음 학기 일부 할인] 도움드릴 수 있는 부분 있으면
말씀 부탁드립니다.

이해 부탁드립니다.
```

### B3. 청약철회 (7일 이내 비대면 결제)

```
[멤버 성함]님 안녕하세요.

전자상거래법 청약철회 기한(결제일로부터 7일) 이내 요청 확인했습니다.
수강을 시작하지 않으신 상태이므로 결제하신 [금액]을
[원 결제수단/입금 계좌]로 [영업일 기준 일정] 내 환불 처리드리겠습니다.

다른 문의 있으시면 말씀 부탁드립니다.
```

## C. 출연료·정산 독촉

### C1. 정중한 1차 독촉 (촬영 후 30일 경과)

```
안녕하세요, [작품명] 출연 건 관련 문의드립니다.

[촬영 종료일] 이후 출연료 정산 일정이 어떻게 진행되고 있는지
확인 부탁드려도 될까요?

계약서상 잔금 지급은 [촬영 종료 후 30일 이내]로 되어 있어서요.
혹시 일정 변동이 있다면 미리 알려주시면 감사하겠습니다.
```

### C2. 강한 2차 독촉 (60일 경과)

```
안녕하세요. 지난번 문의드린 [작품명] 출연료 건입니다.

계약서 제[N]조에 따라 잔금 [금액]원이
[촬영 종료일]로부터 30일 이내 지급되어야 하는데,
현재 [일수]일이 경과한 상태입니다.

이번 주 내로 일정 확인 부탁드립니다.
지급 일정이 불확실한 경우 부득이 다음 단계(내용증명 발송)를 검토할 수밖에 없습니다.

원만한 해결 원하므로 빠른 회신 부탁드립니다.
```

### C3. 내용증명 발송 사전 통지

```
[제작사명] [담당자] 귀하

[작품명] 출연료 잔금 [금액]원 미지급 건에 대하여
[지금까지의 독촉 일자 N회]에도 정산이 이루어지지 않고 있어
부득이 [날짜]까지 지급되지 않을 경우 다음 주 [날짜]에
내용증명을 발송하고 법적 조치를 검토할 예정임을 알려드립니다.

위 일정 이전 정산이 완료되면 별도 조치는 진행하지 않겠습니다.

회신 부탁드립니다.
```

## D. 권리 침해·SNS 분쟁

### D1. 무단 사진/영상 사용 항의 (정중한 1차)

```
안녕하세요, KD4 액팅스튜디오입니다.

[해당 게시물 URL]에 KD4 멤버 [성명]님의 사진/영상이 사용된 것을 확인했습니다.

저희 측에서 사전 사용 동의를 드린 적이 없는 것으로 확인되어
경위 확인 부탁드립니다.

만약 착오 사용이라면 [영업일 N일] 이내 삭제 부탁드리고,
정식 사용을 원하신다면 사용 조건·범위·기간 협의 후 라이선스 진행 가능합니다.

빠른 회신 기다리겠습니다.
```

### D2. 비방글 정정 요청 (사실관계 정정)

```
안녕하세요.

[게시물 URL]에 KD4와 관련해 사실과 다른 내용이 포함되어 있어 정정 요청 드립니다.

▪ 게시된 내용: "[인용 부분]"
▪ 실제 사실: [정정 사실]
▪ 근거 자료: [있다면 첨부]

오해가 있으셨다면 [영업일 N일] 이내 게시물 삭제 또는 정정 부탁드립니다.
이후 추가 조치(내용증명·법적 검토)는 회신 결과 보고 결정하겠습니다.

원만한 해결 원합니다. 감사합니다.
```

## E. 계약 해지·연장

### E1. 해지 사전 통지 (30일 전)

```
안녕하세요.

[계약명] 계약서 제[N]조에 따라 계약 종료일로부터 30일 전 사전 통지를 드립니다.

▪ 계약명: [계약명]
▪ 계약기간: [시작일] ~ [종료일]
▪ 해지 시점: [종료일]
▪ 사유: [선택 — 사업 방향 변경 / 종료 / 미명시 가능]

종료일까지 진행된 [업무/정산]은 정상 진행해주시고,
남은 업무 인수인계 일정은 별도 협의 부탁드립니다.

그동안 협업해주셔서 감사했습니다.
```

### E2. 자동갱신 거절 (해지 통지 기한 직전)

```
안녕하세요.

[계약명] 자동갱신 거절 의사를 미리 전해드립니다.

▪ 계약기간: [현재 기간]
▪ 자동갱신 거절 통지 기한: [날짜] (본 메시지가 해당 기한 내)
▪ 종료 예정일: [현 계약 종료일]

다른 진행 사항은 종료일까지 변동 없이 진행 부탁드리며,
별도 상담 필요하면 말씀 부탁드립니다.

감사합니다.
```

## F. 미성년 멤버 보호자 응대

### F1. 미성년 멤버 출연 동의 요청

```
[보호자 성함]님 안녕하세요. KD4입니다.

[멤버 성함] 학생의 [작품명] 단역 출연 제안이 들어와서 안내드립니다.

▪ 작품: [작품명/매체]
▪ 촬영 일정: [일자] (총 [N]회차)
▪ 출연료: [금액] (세전)
▪ 학교 일정 영향 여부: [평일/주말/방학]
▪ 안전관리: 보호자 동반 가능 + 매니저 동행

미성년 출연이라 부모님 양측 서면 동의가 필요합니다.
참여 의사 있으시면 [연락처]로 회신 부탁드리고,
가능하시면 [날짜]에 미팅 한 번 잡아 자세한 내용 안내드리겠습니다.

감사합니다.
```

### F2. 미성년 멤버 분쟁 1차 응대 (변호사 자문 권유)

```
[보호자 성함]님 안녕하세요.

말씀해주신 내용 잘 확인했습니다.
민감하고 중요한 사안인 만큼 회사 차원에서도 신중하게 사실관계를 확인하고
적절한 조치를 검토하겠습니다.

다만 단독 응대보다 양측 모두 변호사 통한 공식 채널로 진행하는 것이 좋을 듯합니다.
저희도 자문 변호사 통해 회신드리겠으니
[보호자 성함]님께서도 필요하시면 변호사 통해 정식 의견 보내주시기 바랍니다.

[영업일 N일] 이내 1차 회신 드리겠습니다.
```

> ⚠️ 미성년 분쟁은 **단독 응대 절대 금지**. 변호사 거치고, 모든 응대 기록 보존.

## G. 외주·강사 협업

### G1. 외주 단가 협상 (예산 한도 안내)

```
안녕하세요.

견적서 잘 받았습니다. 작업 범위와 결과물 퀄리티 모두 좋아 보입니다.

다만 저희 [프로젝트/예산]의 한도가 [금액]원 정도로 잡혀 있어서요.
견적 [금액]원에서 [금액]원 사이로 조정 가능한지 여쭤봐도 될까요?

조정 어려우시면 [업무 범위 축소] 또는 [회차 분할] 등으로
저희가 맞춰드릴 수 있는 부분도 검토 가능합니다.

회신 부탁드립니다.
```

### G2. 외주 결과물 수정 요청 (계약 범위 내)

```
안녕하세요.

[프로젝트명] 1차 결과물 잘 받았습니다. 전반적으로 좋습니다 :)

계약서 기준 무상 수정 2회 범위에서 다음 사항 수정 부탁드립니다.

1. [수정 사항]
2. [수정 사항]
3. [수정 사항]

수정본 [날짜]까지 회신 부탁드리고,
혹시 위 사항이 합리적 범위를 벗어난다고 판단되시면 미리 말씀 부탁드립니다.

감사합니다.
```

## 메시지 작성 원칙

1. **사실 → 근거 → 요청** 순서
2. **금액·날짜·조항 번호** 구체적으로
3. **존중 어조 + 단호한 입장** 병행
4. **회신 기한** 명시 (없으면 무기한 끌릴 수 있음)
5. **법적 조치 언급은 신중하게** (협박처럼 보이면 역효과)
6. **모든 메시지 보존** (캡처 + 시점 백업)
