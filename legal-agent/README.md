# KD4 법률 에이전트 (Mike 기반)

오픈소스 법률 AI **Mike**(`willchen96/mike`, AGPL v3)를 self-host 하여,
KD4 액팅 스튜디오의 **엔터테인먼트·콘텐츠 법률 + 계약서 검토** 전용 에이전트로 운영하기 위한 문서 모음입니다.

> ⚠️ 이 폴더에는 **문서만** 있습니다. Mike 소스코드는 포함하지 않습니다.
> Mike는 **AGPL v3**라서 KD4(이 repo) 코드와 한 저장소에 섞으면 KD4 전체에 소스 공개 의무가 전염될 수 있습니다.
> 반드시 **별도 저장소/별도 배포**로 운영하세요. (이유는 `SETUP.md` 참고)

## 두 가지 사용 경로

이 에이전트는 두 군데에서 동일한 정체성(`IDENTITY.md`)으로 동작시킬 수 있습니다.

### A) Mike 웹앱 (실제 서비스 — 외부 사용자 대상)
1. `SETUP.md` 따라 self-host 설치. 설치 위치: **`~/Desktop/KD4-HUB/MIKE lawyer/`** (KD4 repo와 분리).
2. Mike 관리자 설정의 시스템 프롬프트 / 커스텀 인스트럭션 영역에 `IDENTITY.md` 내용을 붙여넣기.
3. 사용자는 계약서 업로드 → 검토·요약·독소조항 식별 → 결과 확인.

### B) 이 Claude Code 채팅 (내부 검토용 — 빠른 1차 검토)
이 채팅방을 법률 에이전트로 "켜는" 방법:

```
legal-agent/IDENTITY.md 를 읽고, 그 정체성(KD4 법률 에이전트 '마이크')으로
지금부터 이 대화에 응답해줘. 계약서/법률 질문 위주로만 다뤄.
```

위 한 줄을 새 세션 시작 시 입력하면, 이 채팅이 해당 정체성·면책 규칙을 따릅니다.
(영구 적용을 원하면 세션 시작 훅으로 자동 로드할 수 있습니다 — `SETUP.md`의 "이 채팅에 영구 적용" 절 참고.)

## ⚠️ 반드시 지켜야 할 한계 (내부·외부 공통)

- 이 에이전트는 **변호사가 아니며**, 출력물은 **일반 법률 정보**일 뿐 **법률 자문·대리가 아닙니다.**
- 한국 **변호사법 제109조**: 비변호사가 보수를 받고 법률사무를 취급·알선하면 형사처벌 대상입니다.
  → **외부 유료 서비스로 제공 시 반드시 변호사 감수 체계**를 두거나, "정보 제공 도구"로 명확히 한정하세요.
- 모든 산출물은 **계약 체결·소송 전 반드시 변호사 최종 검토**가 필요합니다.
- AI는 **환각**·최신 법령/판례 미반영 가능. 인용 조문·판례는 **원문 대조 필수**.
- 개인정보·영업비밀 입력 주의. self-host(자가 통제)라도 LLM 제공사로는 데이터가 전송됨을 인지하세요.
