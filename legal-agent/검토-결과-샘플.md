## 샘플 1. 단편 영화 단역 출연계약서 검토

### 의뢰 메시지
> "마이크야, KD4 멤버 김OO 학생이 단편영화 단역 캐스팅됐는데 계약서 검토 부탁해."
> (첨부) 단편영화 'ABC' 출연계약서

### 마이크 응답

> ⚖️ **법률 고지**: 본 답변은 AI가 생성한 일반 법률 정보입니다. 계약 체결 전 변호사 검토를 권장합니다.

**결론**: ⚠️ **수정 후 사인 권장**. 사용권·미편성 시 지급 조항이 멤버에게 불리합니다.

**검토 결과**

| # | 조항 | 현재 내용 요약 | 리스크 | 문제점 | 수정 제안 |
|---|---|---|---|---|---|
| 1 | 제5조 출연료 | "본 작품 방영 후 30일 이내 지급" | **상** | 미편성·미공개 시 영구 미지급 가능 | "촬영 종료일로부터 30일 이내" |
| 2 | 제6조 사용권 | "전 세계·전 매체·영구" | **상** | 광고·머천다이즈까지 무상 사용 | 매체·기간 한정 (국내·10년) + 2차 사용 별도 |
| 3 | 제7조 위약금 | "출연자 일방 부담 1,000만원" | **중** | 균형 결여, 약관규제법 위반 가능 | 양측 균형 + 출연료의 50% 이내 |
| 4 | 제3조 출연료 | "일금 800,000원" | **중** | 세전·세후 표기 누락 | "세전 800,000원 (원천세 3.3% 공제)" 명시 |
| 5 | 제2조 일정 | 회차 ⑥ 빈칸 | **하** | 모호함 → 분쟁 시 입증 곤란 | "회당 200,000원 × 4회" 명기 |
| 6 | 제10조 관할 | "제작사 주소지 부산지방법원" | **하** | 멤버 부담 ↑ | "서울중앙지방법원" 또는 합의관할 |

**탑3 — 반드시 짚을 것**
1. 사용권 범위 → 한정 (국내·매체 한정·10년)
2. 지급 기준 → 촬영 종료일 기준
3. 위약금 균형 → 양측 동등

**다음 단계 (카톡 메시지)**
```
안녕하세요, 'ABC' 단편영화 출연계약서 검토 잘 받았습니다.

몇 가지 사항만 수정해주시면 바로 사인해서 회신드리겠습니다.

1. 제5조 잔금 지급 — '촬영 종료일로부터 30일 이내'로 변경 부탁드립니다.
2. 제6조 사용권 — 매체와 기간 한정 (국내, 본편/예고편/홍보 클립, 10년) 부탁드리고,
   광고·머천다이즈 등 2차 사용은 별도 합의로 분리해주시면 좋겠습니다.
3. 제7조 위약금 — 양 당사자 균형 차원에서 제작사 측 위약금 조항도 추가 부탁드립니다.
4. 제3조 출연료 — '세전 80만원 (원천세 3.3% 공제)' 명기 부탁드립니다.
5. 제2조 회차 빈칸 — '회당 20만원 × 4회 = 합계 80만원' 채워주시면 감사하겠습니다.

수정본 받는 대로 바로 회신드리겠습니다. 감사합니다.

**추가 확인 필요**
- 제작사 사업자등록증 사본 (KYC)
- 미성년 멤버 → 친권자 양측 동의 필요

> ⚖️ **법률 고지**: 본 답변은 AI가 생성한 일반 법률 정보이며, 법률 자문이나 변호사의 대리가 아닙니다.
> 실제 계약 체결·분쟁·소송 전에는 반드시 변호사의 검토를 받으시기 바랍니다.

## 샘플 2. 환불 분쟁 응대

### 의뢰 메시지
> "멤버 한 명이 수강 시작 3주 됐는데 갑자기 전액 환불 요구해. 8주짜리 클래스고 결제는 80만원. 어떻게 응대해야 돼?"

### 마이크 응답

> ⚖️ **법률 고지**: 본 답변은 AI가 생성한 일반 법률 정보입니다.

**결론**: 학원법 환불 기준 적용 시 **약 26만원 환불 가능** (전액 환불 의무 없음).

**산정**
- 총 수강시간: 8주
- 진행 회차: 3주차 → 진행률 약 37.5%
- 학원법 시행령 별표 4 적용:
  - 총 수강시간의 1/3(약 2.7주) 경과 후, 1/2(4주) 경과 전 → **이미 납부한 수강료의 1/2 환불**
- 80만원 × 1/2 = **40만원 환불**

> ※ 약관에 더 유리한 환불 비율을 적용하기로 명시되어 있다면 그를 따름. 위는 학원법 기준 최소치.

**응대 카톡**
```
[멤버 성함]님 안녕하세요. KD4입니다.

환불 요청 잘 확인했습니다.
저희 약관과 학원법 시행령 기준에 따라 환불 비율 안내드립니다.

▪ 결제일: [날짜]
▪ 결제 금액: 800,000원
▪ 개강일: [날짜]
▪ 현재 진행: 3주차 (총 8주)
▪ 진행률: 약 37.5% (총 수강시간의 1/3 경과 후 1/2 경과 전 구간)

학원법 환불 기준:
- 이 구간 환불 비율: 이미 납부한 수강료의 1/2
- 환불 가능 금액: 800,000원 × 1/2 = **400,000원**

위 금액은 환불 요청일로부터 5영업일 이내 [입금 계좌]로 입금드립니다.
이의 있으시면 회신 부탁드립니다.
```

**거절·확전 대비**
- 멤버가 전액 환불 강하게 주장 시 → **약관 사본 + 학원법 시행령 별표 4** 인용 응대
- 소비자원 분쟁조정 신청 시 → 회사 정당성 입증 가능 (협력)
- SNS 폭로 협박 시 → **무시 + 기록 보존** (협박에 굴복 시 다른 멤버 형평 문제)

**KD4 약관 점검 권장**
- 현재 약관에 환불 조항이 학원법 기준과 일치하는지 확인
- 일치하지 않으면 → `legal-agent/templates/클래스-수강-약관.md` 참고하여 갱신

> ⚖️ **법률 고지**: 본 답변은 일반 법률 정보입니다. 분쟁이 확전되면 변호사 자문 권장.

## 샘플 3. 위장 프리랜서 검토

### 의뢰 메시지
> "우리 마케팅 보조 친구를 프리랜서 3.3%로 쓰고 있는데, 매주 화요일, 목요일 11시-17시 회사 출근하고 일하는 형태야. 문제 있어?"

### 마이크 응답

> ⚖️ **법률 고지**: 본 답변은 일반 법률 정보입니다.

**결론**: ⚠️ **위장 프리랜서 가능성 매우 높음**. 단시간 근로계약서로 전환 권장.

**근로자성 판단 체크**
- ✅ 고정된 출퇴근 시간 (매주 화·목 11~17시)
- ✅ 고정된 출근 장소 (KD4 사무실)
- ✅ KD4의 업무 지휘·감독 (마케팅 보조)
- ✅ 보수가 근무시간에 비례 (시급·일급으로 환산 가능)
- ❌ 다른 거래처 자유 (확인 필요)

→ **위 5개 중 4개 ✅** → 근로자 인정 가능성 높음

**사후 재분류 시 위험**
- 4대 보험 소급 가입 + 가산세 (수년치)
- 연차·주휴수당 소급 지급 (단, 주 15시간 미만이면 면제 가능)
- 퇴직금 (1년 이상 근속 시)
- 부당해고 구제 신청 가능 (5인 이상 사업장)

**다행스러운 점**
- 주 11시간(5.5h × 2일) 근로 → **주 15시간 미만** → 주휴·연차·퇴직금 발생 ❌
- 5인 미만 사업장 → 가산수당·연차·부당해고 구제 미적용
- **남는 위험은 산재**

**권장 조치**
1. **단시간 근로계약서**로 전환 (양식: `templates/단시간-근로계약서.md`)
2. **산재보험 가입** (근로복지공단, 사용자 100% 부담, 임금의 약 0.7%)
3. 4대 보험 중 고용·연금·건강은 월 60시간 미만이면 미가입 가능
4. 원천징수: 사업소득 3.3% → **근로소득 간이세액표** (단시간 저액이면 사실상 0원)

**비교 (현재 vs 권장)**

| 항목 | 현재 (프리랜서 3.3%) | 권장 (단시간 근로자) |
|---|---|---|
| 세금 | 사업소득 3.3% | 근로소득 간이세액 (저액 0원) |
| 4대보험 | 산재 0 | 산재 만 가입 (사용자 부담 약 7천원/월) |
| 위장 위험 | 매우 높음 | 없음 |
| 분쟁 가능성 | 높음 | 낮음 |
| 추가 비용 | 미가입 가산세 위험 | 월 7천원~ |

**결론**
- 추가 부담 월 약 1만원 미만
- 위장 프리랜서 위험 완전 제거
- **단시간 근로계약서 + 산재 가입** 적극 권장

> ⚖️ **법률 고지**: 본 답변은 일반 정보입니다. 정식 전환 전 공인노무사 자문 권장.

## 샘플 4. 비방글 대응

### 의뢰 메시지
> "전 멤버가 인스타에 '@kd4actingstudio 수업료만 비싸고 강사 실력 별로'라고 올렸어. 명예훼손으로 고소 가능?"

### 마이크 응답

> ⚖️ **법률 고지**: 본 답변은 일반 법률 정보입니다.

**결론**: ⚠️ **형사 고소는 신중**. 사실적시 명예훼손 성립 어려우며, 역공 위험 있음.

**법적 분석**

해당 게시물 분석:
- "수업료만 비싸고" → **의견 표명** (가격이 본인 기준에 비해 비싸다고 느꼈다는 평가)
- "강사 실력 별로" → **의견 표명** (주관적 평가)

**명예훼손 성립 요건**
- ✅ 공연성 (SNS 공개 게시)
- ❌ **사실의 적시** — "비싸다·별로다"는 사실이 아닌 의견·평가
- △ 사회적 평가 저하 — 한정적
- ✅ 특정성 (@kd4actingstudio 태그)

→ **사실 적시 명예훼손 미성립 가능성 높음**
→ 모욕죄 검토 가능하지만 단순 평가에는 미성립 多

**위험**
- 무리한 형사 고소 시 **무고죄 역공** 위험
- 자칫 "악덕 학원이 비판 멤버 고소" 프레임 → 평판 손상
- 언론·SNS 확전 가능

**권장 대응 (확전 회피)**

```
1단계: 사실 확인
   - 본 비판이 일부라도 사실 근거 있는가 (수업료·강사 평가)
   - 일부 사실이면 → 내부 개선 우선. 법적 대응 ❌

2단계: 정중한 대화 시도
   - 본인에게 직접 카톡 1차 응대
   - "어떤 부분이 아쉬우셨는지 자세히 들려주시면 개선하겠다"
   - 불만 해소 → 자발적 삭제 가능성

3단계: 게시물 정면 반박 (선택)
   - KD4 공식 SNS에 가격·강사 정보 공개 자료 게시
   - 다른 멤버 만족도·후기 자연스럽게 노출
   - 비방글에 직접 반응 ❌

4단계: 법적 검토 (마지막)
   - 비방이 허위 사실(예: "강사가 폭언했다" 등) 포함 시
   - 변호사 통해 정중한 정정 요청 내용증명
   - 그 후에도 미시정 시 형사·민사 검토
```

**참고 — 명예훼손 vs 모욕 vs 영업방해**

| 죄목 | 요건 | 본 사안 적용 |
|---|---|---|
| 사실적시 명예훼손 | 사실의 적시 + 사회적 평가 저하 | △ (사실 적시 부족) |
| 허위사실 명예훼손 | 허위 사실 + 사회적 평가 저하 | ❌ (허위 사실 없음) |
| 모욕 | 모욕적 언사 (욕설 등) | ❌ (욕설 없음) |
| 업무방해 | 위계·위력으로 업무 방해 | ❌ (의견 표명) |

→ **현재 게시물은 형사 처벌 대상 어려움**

> ⚖️ **법률 고지**: 본 답변은 일반 정보입니다. 실제 형사 고소 검토 시 반드시 변호사 자문.

## 샘플 5. 외주 영상편집 비용 협상

### 의뢰 메시지
> "외주 영상편집팀 견적이 회당 250만원이래. 우리 예산은 150만원. 어떻게 협상?"

### 마이크 응답

> ⚖️ **법률 고지**: 본 답변은 일반 정보입니다.

**결론**: 단가 협상 + 업무 범위 조정으로 150~180만원 합의 가능성 있음.

**협상 전략 옵션**

**옵션 A — 단가 직접 조정 요청**
```
견적 250만원 → 180만원 (28% 인하)
명분: 정기 발주 약속 (월 N건) + 장기 거래 관계
```

**옵션 B — 업무 범위 축소**
```
원안 250만원 (촬영·편집·자막·BGM·납품 형식 3종)
↓
150만원 (편집·자막·BGM·납품 1종)
- 촬영 별도 위탁
- 납품 1종으로 축소 (4K MP4만)
- raw 소스 공유 (KD4가 추가 활용)
```

**옵션 C — 분기 정기 발주 (단가 ↓ + 안정성 ↑)**
```
1건 단가 250 → 1년 단가 X 8건 = 1,440만원 (월 평균 120만원)
- 발주 안정성 제공
- 외주팀: 매출 예측 가능 → 단가 양보
```

**카톡 카운터 (옵션 A·C 조합)**
```
안녕하세요.
