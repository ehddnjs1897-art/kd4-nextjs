# 계약서 검토 표준 체크리스트 (마이크 워크플로)

> **사용법**: 계약서 검토 의뢰가 들어왔을 때 이 체크리스트 순서대로 짚는다.
> 각 항목별 "리스크 등급(상/중/하) + 수정 제안" 형태로 결과를 정리한다.

---

## 1단계 — 메타 정보 확인 (10초)

| # | 체크 항목 | 확인 |
|---|---|---|
| 1 | 계약 유형이 무엇인가? (출연·전속·매니지먼트·외주·NDA·근로·임대차…) | ☐ |
| 2 | 당사자가 누구인가? (상호·대표자·사업자등록번호 일치 확인) | ☐ |
| 3 | 한쪽 당사자 표기가 본문 곳곳에서 일관된가? (별칭·옛 상호 혼용 주의) | ☐ |
| 4 | 계약일·서명일이 명기되어 있는가? | ☐ |
| 5 | 계약서 페이지 누락·간인 누락이 없는가? | ☐ |

> 💡 KD4에서 자주 본 사고: **제작사 등기상호 vs 사업자등록증 상호 vs 통상 사용명** 불일치
> → 등기·사업자등록증 모두 확인. 분쟁 시 누가 책임지는지 불명확해짐.

---

## 2단계 — 본문 핵심 조항 (조항별 체크)

### A. 계약기간·갱신·해지

- [ ] 시작일·종료일 명시 (또는 "기간의 정함이 없음")
- [ ] **자동갱신 조항**: "이의 없으면 동일조건 자동연장" → KD4에게 일방 불리한지 확인
- [ ] 해지 사전통지 기간 (통상 30일~3개월)
- [ ] 즉시해지 사유 명확화 (모호한 "신뢰관계 파탄" 단독 사유는 위험)
- [ ] 해지 시 정산 방법 (선급금 반환·잔여 회차 보수 등)

### B. 보수·정산·지급시기

- [ ] **세전/세후** 명확 표기
- [ ] 원천징수 종류 (사업소득 3.3% / 근로소득 간이세액 / 기타소득 8.8%)
- [ ] 지급일 (촬영 종료 기준? 방영 기준? 정산 기준?)
  - ⚠️ "방영 후 30일" → 미편성·결방 시 영구 미지급 위험. **촬영 종료 기준**으로 변경 요구
- [ ] **지급 지연 시 이자** 조항 (연 6%~연 12% 상사법정이자 등)
- [ ] 부가가치세 별도/포함
- [ ] 분할 지급의 경우 각 회 명확화 (착수금·중도금·잔금 비율)

### C. 권리귀속 (저작권·초상권·퍼블리시티권)

- [ ] **저작재산권 양도 vs 사용 허락** 구분
- [ ] 양도라면 **양도 시점**(검수 완료? 대금 완납?) 명시
- [ ] 사용권이라면 **지역·매체·기간·횟수** 4가지 모두 한정
  - 위험 신호: "전 세계·모든 매체·영구·무제한"
- [ ] **2차적저작물 작성권**: 별도 명시 없으면 양도되지 않음 (저작권법 제45조 제2항)
- [ ] **저작인격권**: 양도 불가 (제14조). 행사 제한 합의는 가능
- [ ] **퍼블리시티권/초상권**: 광고·머천다이즈 등 상업적 이용은 별도 합의·추가 보수

### D. 업무 범위·전속의무·경업금지

- [ ] 업무 내용이 구체적으로 특정되었는가? (광범위 정의 → 추가 보수 없이 일감 폭증)
- [ ] **전속의무**: 동종 업무 금지 → 기간·지역·업종 범위 합리적인가?
- [ ] **경업금지**(퇴직 후): 6개월~1년 + 동일 권역 정도가 합리적. 2년 이상 + 전국은 무효 위험
- [ ] 경업금지에 대한 **반대급부**(보상금) 있는가?

### E. 위약금·손해배상

- [ ] **양 당사자 균형**: 한쪽에만 과도한 위약금은 약관규제법 위반 가능
- [ ] **위약벌 vs 손해배상예정**: 위약벌은 손해 입증 무관, 손해배상예정은 별도 손해 청구 제한
- [ ] **책임제한**: 상한선(예: 계약금 3배) 있는가? 고의·중과실 예외인가?
- [ ] **간접손해 제외** 조항 (일실이익·평판 손해 등)

### F. 비밀유지·개인정보

- [ ] 비밀정보 범위 명확화
- [ ] 비밀유지 기간 (계약 종료 후 2~5년)
- [ ] 위반 시 손해배상예정액
- [ ] 개인정보 처리 시 개인정보보호법 준수 조항

### G. 준거법·관할·중재

- [ ] 준거법: 대한민국 법 (해외 거래 시 특히 중요)
- [ ] 관할법원: 어느 쪽 주소지인가? 합의관할 가능한가?
- [ ] 중재 합의 시: 중재기관·중재지·중재인 수
  - ⚠️ 중재는 단심제·소송 불가. 신중히 동의

### H. 기타 표준 조항

- [ ] 양도 금지 (지위 양도 시 사전 동의)
- [ ] 통지 방법 (이메일·서면)
- [ ] 분리가능성 (일부 무효 시 나머지 유효)
- [ ] 완전합의 (구두 합의 무효화)
- [ ] 수정·변경은 서면으로만

## 3단계 — 산업별 특칙 체크

### 엔터테인먼트·콘텐츠

- [ ] **표준전속계약서** 준수 여부 (대중문화예술인 표준전속계약서 — 문체부 고시)
  - 전속 7년 한도, 정산 명세 제공 의무, 부당 시 일방해지권 등
- [ ] **저작인접권**(실연자): 복제·배포·전송권 → 양도/허락 구분
- [ ] **음악 사용**: 한국음악저작권협회(KOMCA)·한국음악실연자연합회·한국음반산업협회 처리 명확화
- [ ] **영상저작물 특례** (저작권법 제100조): 영상저작물 이용에 필요한 권리는 영상제작자에게 양도 추정
- [ ] **미성년 출연**: 친권자 모두 동의·근로시간·학습권 보장

### 광고·마케팅

- [ ] **표시광고법**: 부당광고·과장광고 금지 → 보장·효과 단정 표현 주의
- [ ] **뒷광고**: 경제적 이해관계 표시 의무 (#광고 #협찬)
- [ ] **모델 출연료 vs 광고주 책임 분리**: 광고 내용 위법 시 모델 책임 한정

### 근로·인사

- [ ] **표준근로계약서** 사용 여부
- [ ] **임금명세서 교부 의무** (근로기준법 제48조 제2항)
- [ ] **주 15시간 미만**: 주휴·연차·퇴직금 면제
- [ ] **5인 미만 사업장**: 가산수당·연차·부당해고 구제 미적용
- [ ] **4대 보험** 가입 의무 (산재는 무조건 의무)

### 개인정보·정보보호

- [ ] **수집·이용 동의** (목적·항목·기간 명시)
- [ ] **제3자 제공 동의** 별도
- [ ] **마케팅 활용** 별도 선택 동의
- [ ] **보유기간 만료 후 파기** 의무
- [ ] **개인정보 위탁 시** 수탁자·위탁업무·관리감독 의무

## 4단계 — 산출물 포맷

검토 결과를 다음 표로 정리한다.

| # | 조항 | 현재 내용 요약 | 리스크 등급 | 문제점 | 수정 제안 |
|---|---|---|---|---|---|
| 1 | 제5조 출연료 | "방영 후 30일 이내 지급" | **상** | 미편성 시 영구 미지급 위험 | "촬영 종료 후 30일 이내"로 변경 |
| 2 | 제6조 사용권 | "전 세계·전 매체·영구" | **상** | 광고·머천다이즈 무제한 사용 우려 | 매체·기간 한정 + 별도 보수 |
| ... | | | | | |

**리스크 등급 기준**:
- **상**: 즉시 수정 요청. 미수정 시 계약 거절 검토
- **중**: 수정 권장. 수정 거부 시 별도 부속합의로 보완
- **하**: 표준적 관행. 그대로 진행 가능하나 인지

## 5단계 — 마무리 코멘트 (사용자 대상 안내)

1. **결론 한 줄**: "그대로 사인 OK / 일부 수정 후 사인 / 사인 보류 권장"
2. **반드시 짚을 3가지** (탑3 독소조항 또는 우려 사항)
3. **추가 확인이 필요한 사실 관계** (예: 제작사 사업자등록증 확인)
4. **변호사 검토 권고 여부** (금액·복잡성·전속기간 기준)
5. **면책 고지** (`IDENTITY.md` 제6조)

## 면책

> ⚖️ **법률 고지**: 본 체크리스트는 일반 실무 참고용이며, 개별 사안의 법률 자문이 아닙니다.
> 계약금액 1,000만원 이상, 전속기간 1년 이상, 권리 양도 포함 계약은 변호사 검토를 권장합니다.
