## A. 멤버·인사·노무

### A1. 파트타임 멤버를 프리랜서(3.3%)로 쓰면 안 되나?

**짧은 답**: 사실관계가 **근로자**에 가까우면 안 됨.

**판단 기준** (대법원 사용종속성 기준 — 종합 판단)
- 고정된 출퇴근 시간·장소 ✅ → 근로자 가까움
- 회사의 업무 지시·감독 ✅ → 근로자 가까움
- 보수가 근무시간(시급·일급·월급)에 비례 ✅ → 근로자 가까움
- 다른 거래처 일 제한 ✅ → 근로자 가까움
- 비품·재료 회사 부담 ✅ → 근로자 가까움

→ 3개 이상이면 **위장 프리랜서**. 사후 근로자로 인정 시:
- 4대 보험 소급 가입 + 가산세
- 연차·주휴수당·퇴직금 소급 지급
- 부당해고 구제 신청 가능

**KD4 권고**: 출근 통제하는 멤버는 처음부터 **단시간 근로계약서** 사용.
주 15시간 미만으로 운영하면 주휴·연차·퇴직금 부담 없음.

### A2. 5인 미만 사업장은 근로기준법 적용 안 되는 거 아닌가?

**전부 적용 제외 아님**. 일부만 제외.

| 적용 | 적용 제외 |
|---|---|
| 근로계약서 서면 교부 (제17조) | 가산수당 50% (제56조) |
| 임금명세서 교부 (제48조) | 연차유급휴가 (제60조) |
| 최저임금 | 부당해고 구제신청 |
| 해고예고 30일 (제26조) | |
| 산재보험 의무 | |
| 주휴수당 (제55조) | |

**즉**: 5인 미만이라도 근로계약서·임금명세서·산재·해고예고는 의무.
가산수당(연장·야간·휴일근로)과 부당해고 구제만 면제.

### A3. 산재는 어떻게 가입하나?

1. **근로복지공단** 홈페이지(www.kcomwel.or.kr)에서 신고
2. 사업장 최초 가입 → "보험관계 성립신고서" 제출 (사업 개시 14일 이내)
3. 매년 3월 31일까지 "보수총액신고서" 제출 (전년도 임금 신고)
4. 보험료 = 임금총액 × 업종별 요율 (학원·교육업: 약 0.7%)
5. **전액 사용자 부담**, 근로자 공제 없음

**미가입 상태에서 산재 발생 시**:
- 근로자에게 산재보험 급여는 정상 지급
- 사용자는 미가입 가산금 + 보험료 소급 부담
- 산재 발생 자체에 대한 형사처벌 별도 가능

### A4. 4대 보험을 안 들면 비용이 얼마나 절약되나?

| 보험 | 본인 부담 | 사용자 부담 |

| 국민연금 | 4.5% (45,000) | 4.5% (45,000) |
| 건강보험 | ~3.5% (35,000) | ~3.5% (35,000) |
| 고용보험 | 0.9% (9,000) | 1.05%~ (10,500) |
| 산재보험 | 0 | 업종별 (학원: 약 7,000) |

**사용자 총 부담**: 약 97,500원 (임금의 9.75%)
**근로자 실수령 감소**: 약 89,000원

→ 단시간 근로(월 60시간 미만) → 산재만 가입 시 약 7,000원/월 부담

### A5. 강사를 정규직 vs 외주(프리랜서)로 쓸 때 차이?

| 항목 | 정규직 (근로자) | 외주 (프리랜서) |
|---|---|---|
| 세금 | 근로소득 간이세액 | 사업소득 3.3% |
| 4대 보험 | 의무 (산재 포함 4개) | 산재 미가입 (단, 예술인 고용보험 가입 가능) |

## B. 출연·실연·저작권

### B1. 멤버가 영화 단역 출연했는데 권리 침해 같다. 어떻게 대응?

**확인 사항**
1. 출연계약서 사본 보유 여부
2. 계약상 "사용 범위" 조항 (지역·매체·기간)
3. 실제 사용된 범위 vs 계약된 범위 비교
4. 침해 증거 확보 (스크린샷·URL·방영 일시)

**대응 단계**
1. **내용증명**: 침해 중단 + 손해배상 청구
2. **저작권위원회 분쟁조정** (저작권법 제112조)
3. **민사 가처분** (사용 중단 + 손해배상)
4. **형사 고소** (저작권법 제136조 — 5년 이하 징역 또는 5천만원 이하 벌금)

**합의금 산정 기준 (실무)**
- 통상 사용료 × 침해 횟수·기간
- 부당이득 산정
- 위자료(인격권 침해 시)

---

### B2. KD4 클래스 영상을 회사 SNS에 올리려면 멤버 동의 어떻게 받나?

**필요한 동의 항목**
1. 촬영 자체 동의 (초상권)
2. 게시 매체 (SNS·홈페이지·교재) 동의
3. 게시 기간
4. 본인 성명·예명 표기 여부

**동의 받는 방법**
- 수강 신청서에 **선택 동의 항목**으로 포함 (필수 동의로 강제 ❌)
- 별도 "초상권 사용 동의서" 작성
- 가능하면 매번 영상 단위로 추가 동의 (특히 발표회·공연)

**미성년 멤버**: **법정대리인 동의** 필수

**철회권**: 멤버가 사후 게시 중단 요청 시 → 14일 이내 처리 의무

---

### B3. 강사 메소드를 KD4 커리큘럼으로 사용하려면?

**저작권법 관점**
- 강사가 직접 만든 교재·영상은 **강사 저작물**
- 영업비밀(노하우)은 부정경쟁방지법 보호 대상

**KD4가 권리를 가지려면**
1. **양도 계약**: 강사 → KD4 저작재산권 양도 (대가 지급)
2. **이용 허락**: 강사 보유, KD4 사용권 (기간·범위 한정)
3. **업무상저작물**: 강사가 KD4 근로자 + 업무 범위 내 작성 → KD4 귀속(저작권법 제9조)
   ⚠️ 외주·프리랜서 강사는 업무상저작물 아님

**권고**: 외주 강사에게는 **이용 허락 + 강사 퇴직 후 사용 기간** 명시.

---

### B4. 멤버가 SNS에 KD4 수업 사진을 자유롭게 올려도 되나?

**기본 원칙**
- 본인 사진 → 본인 SNS는 자유
- 다른 멤버·강사 함께 찍힌 사진 → **타인 동의** 필요
- KD4 시설·교재 노출 → 영업비밀 보호 차원에서 사전 동의 권장

**KD4 약관에 명시할 것**
- "수업 중 무단 촬영·녹음 금지" + "본인 출연 영상도 다른 멤버·강사 포함 시 동의 필요"
- 위반 시 수강 자격 정지 가능

## C. 광고·마케팅·인플루언서

### C1. 멤버가 KD4 추천 인스타 포스팅하면 #광고 표시 의무?

**기준**: 멤버가 **경제적 이해관계**(현금·할인·우대) 받았는지

| 상황 | #광고 표시 의무 |
|---|---|
| 무료 클래스 제공받고 후기 | **의무** |
| 추천 보상 받고 추천 글 | **의무** |
| 본인 비용으로 수강하고 자발적 후기 | 의무 없음 |

**위반 시**: 표시광고법 + 공정거래위원회 가이드라인 위반
→ 최대 5억원 과징금 + 시정명령

**권고 문구**
- "#광고 #협찬 #KD4지원 #PR"
- 영상이면 첫 화면 노출 + 자막
- "본 게시물은 KD4로부터 [할인/지원]을 받고 작성되었습니다"

---

### C2. KD4 광고에 "100% 합격" "최단기 데뷔" 같은 문구 써도 되나?

**위험**: 표시광고법 위반 (부당광고 — 거짓·과장·기만·비교)

**금지되는 표현**
- "100% 합격 보장"
- "업계 1위" (객관적 근거 없으면)
- "00 대기업 출신 강사" (대표성·재직 여부 미확인 시)
- "최단기간 데뷔" (객관 기준 없음)

**허용되는 표현**
- "지난 3년간 멤버 데뷔 사례 N건" (사실 + 출처 표시)
- "OO 출신 강사 N명 재직" (사실 확인)
- "프리미엄 코칭 시스템" (주관적 평가 — 단정적이지 않으면)

**위반 시**: 공정거래위원회 시정명령·과징금. 경쟁사 신고로 적발 잦음.

---

### C3. 멤버 후기를 광고에 쓰려면?

**필요한 것**
1. 멤버의 **서면 동의** (사용 매체·기간·범위)
2. **사실 확인** (조작·과장 ❌)
3. 후기 끝에 "**개인의 경험·소감이며 효과는 개인차가 있습니다**" 고지
4. 보상받은 후기는 #광고 표시 (위 C1)

**금지**
- 직원·관계자 후기를 일반 멤버 후기로 위장
- 후기 일부 발췌해서 의미 왜곡
- 실재하지 않는 멤버 가공 후기

## D. 개인정보·정보보호

### D1. 멤버 DB 운영하는데 개인정보보호법상 의무?

**KD4가 처리하는 개인정보 예시**
- 이름·연락처·이메일·생년월일 (수강 신청)
- 결제 정보 (카드번호·계좌)
- 영상·사진 (수업 녹화·발표회)
- 마케팅 동의 정보

**필수 의무**
1. **수집·이용 동의** (목적·항목·보유기간 명시)
2. **제3자 제공 동의** (캐스팅 매칭 시 제작사 제공 등)
3. **마케팅 활용 동의** 별도
4. **개인정보처리방침 공개** (웹사이트 하단 링크)
5. **개인정보보호책임자 지정** (CPO)
6. **보유기간 만료 후 파기** 의무
7. **개인정보 유출 시 신고** (72시간 이내, 개인정보보호위원회 + 본인)

**미준수 시**: 최대 3% 매출 과징금 + 형사처벌

### D2. 카카오톡으로 멤버에게 광고 메시지 보내도 되나?

**필수 조건** (정보통신망법 제50조)
1. **사전 수신 동의** (옵트인)
2. 메시지 첫 부분 "[광고]" 표시
3. 광고주 명·연락처 표기
4. **수신 거부 방법** 안내 (1회 클릭으로 해제 가능해야)
5. 거부 시 즉시 발송 중단

**예외**: 거래 관계 유지 안내 (수강 일정·환불 안내 등 — 광고 아님)

**위반**: 3,000만원 이하 과태료

### D3. 멤버가 "내 정보 삭제해줘"라고 요청하면?

**개인정보보호법 제36조 — 처리정지·삭제 요구권**

**대응 절차**
1. 본인 확인 (신분증·인증)
2. 10일 이내 처리
3. 결과 통지

**거부 가능 사유**
- 법령상 보유 의무 (세법·전자상거래법 등 5년)
- 계약 이행을 위한 보유 (수강 종료 후 일정 기간)
- 다른 사람의 권리·이익 보호 필요

**KD4 권고**: 약관에 "보유기간 분리" 명시
- 수강 정보: 종료 후 3년
- 결제 정보: 5년 (세법)
- 마케팅 동의 정보: 동의 철회 시 즉시 파기

## E. 사업·세무·자금

### E1. 멤버에게서 받은 수강료 영수증·세금계산서 발급 의무?

**원칙**: 영수증 발급 의무 있음 (소득세법)

**구분**
- 개인 멤버 → 현금영수증 또는 카드 매출전표
- 사업자 멤버 (회사 비용 처리) → **세금계산서** 요청 시 발급 의무 (10만원 이상 거래)

**미발급 시 가산세**: 미발급 금액의 2% (현금영수증), 1~2% (세금계산서)

### E2. KD4 매출 1억 5천이면 어떤 세무 의무?

**부가가치세** (일반과세자 기준)
- 매년 1, 7월 신고·납부
- 매출 세액 - 매입 세액 차이 납부
- 면세 (학원·교육) 적용 가능성 확인 필요

**종합소득세**
- 매년 5월 신고
- 사업 결산 후 순이익에 대해 누진세율
- 1억 5천 매출 + 비용 1억 → 순이익 5천만원 → 약 600~700만원 세금

**4대 보험** (사업주 본인)
- 국민연금 (월 51만원 한도)
- 건강보험 (지역가입자)
- 산재 (1인 이상 직원 시)

**성실신고 대상**
- 서비스업 매출 5억 이상 → 세무사 확인 의무
- 1억 5천은 미해당

### E3. 정산을 카카오톡으로 한 약속은 효력 있나?

**유효함**.

**증거 가치**
- 카톡 대화 = 전자적 의사표시 (전자문서법)
- 화면 캡처·기록 보존 → 민사·형사 증거 채택
- 발신자 본인 확인 (본인 명의 카톡 + IP·기기 정보)

**다만 한계**
- 모호한 표현 ("나중에 잘 챙겨드릴게요") → 구체성 부족 시 불리
- 가능하면 **금액·시기·조건 명확하게** 카톡으로 정리

**권고**: 중요한 합의는 **부속합의서·각서** 형태로 PDF 서명. 카톡은 보조 증거.

## F. 위기 대응

### F1. 멤버가 "환불 안 해주면 신고한다"고 협박할 때

**침착하게**
1. 약관·결제 시점 확인
2. 학원법 제18조 환불 기준 적용 시 환불 비율 계산
3. 환불해야 하면 즉시 환불 + 분쟁 종결
4. 환불 의무 없거나 일부만 의무면 **약관 근거 + 환불 비율** 안내 서면 발송
5. **소비자원 분쟁조정** 신청 시 협조 (강제력 없으나 평판 영향)
6. 사실관계가 맞으면 **소비자원 권고** 따르는 게 좋음

**금지**
- 협박에 굴복해 약관 위반 환불 (다른 멤버 형평 문제)
- 협박이라고 모욕·명예훼손 응대 (역공 위험)
- 카톡·SNS에서 멤버 비방 (정보통신망법)

---

### F2. 강사·전 멤버가 KD4 비방글을 SNS에 올렸을 때

**대응 단계**
1. **사실관계 확인**: 비방글 내용이 사실인지 vs 허위인지
2. **사실이면**: 내부 개선 + 진심 어린 사과. 법적 대응 비추천 (역효과)
3. **허위·왜곡이면**:
   a. 스크린샷 보존 + 게시일·URL 기록
   b. 작성자에게 정중하게 정정·삭제 요청 (내용증명)
   c. 무응답·거부 시 **명예훼손·정보통신망법** 검토
   d. 플랫폼에 임시조치(블라인드) 신청

**형사**: 정보통신망법 제70조 (사실 적시 명예훼손 3년 / 허위 7년 이하)
**민사**: 명예훼손·업무방해 손해배상

**권고**: 변호사 자문 + 내용증명 → 가처분 → 본안 순서.

---

### F3. 미성년 멤버 부모가 법적 분쟁 걸어왔을 때

**즉시 조치**
1. 멤버 안전 확보 (성희롱·체벌·학대 의혹은 별도)
2. 사실관계 기록 (수업 일지·영상·증인 진술)
3. **변호사 자문 즉시 요청** (지체하면 불리)
4. 부모에게 **단독 응대 금지** (반드시 변호사 또는 공동 응대)
5. 보험사 통지 (영업배상책임보험 가입 시)

**미성년 관련 분쟁은 특수성**
- 사회적 비난·언론 보도 위험 큼
- 회사 책임 인정 시 더 큰 분쟁 회피 가능
- 모든 응대 기록 보존

## G. 자주 묻는 잡다한 질문

### G1. 영어 학원과 협업하는데 멤버 정보 공유해도 되나?
→ **개인정보 제3자 제공 동의** 필수. 동의 없으면 위법.

### G2. 클래스 영상 유튜브에 광고 띄워도 되나?
→ 강사·멤버 출연자 동의 + 음악 저작권 처리(KOMCA 라이선스) 필요.

### G3. 데뷔한 멤버가 "KD4 출신"이라고 홍보해주는 게 가능한가?
→ 본인 동의 받으면 가능. 단, "KD4 출신" 사용권 명시 + 보상 협의.

### G4. 외국인 멤버·강사 채용 시 비자 문제?
→ E-6(예술흥행)·D-10(구직) 등 비자 종류 확인. **법무부 출입국·외국인정책본부** 또는 행정사 자문.

### G5. 회사 명의로 차량 사고 났을 때 멤버 부상 책임?
→ 영업배상책임보험·자동차종합보험 적용 범위 확인. 운전자 사적 사용 여부 쟁점.
