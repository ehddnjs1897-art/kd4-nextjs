# 합의서·각서 양식 모음

> **합의서**: 분쟁 종결·새로운 합의 도달 시 작성. 법적 구속력 있음.
> **각서**: 일방의 약속·확약. 본인 서명만 있어도 효력.
> **공증** 필요: 강제집행이 예상되거나 금액이 크면 (공증사무소 or 공증인가 법무법인).

---

## 양식 1. 분쟁 종결 합의서 (포괄)

```
                          합 의 서

[당사자 A] (이하 "갑")와 [당사자 B] (이하 "을")는 [분쟁 사안 요약]에 관하여
원만한 해결을 위하여 다음과 같이 합의한다.

제1조 (합의 내용)
   ① 갑은 을에게 합의금 [____원]을 본 합의서 서명일로부터 7일 이내
      을 명의 ________________________ 계좌로 지급한다.
   ② 을은 위 합의금 수령을 조건으로 본 사안과 관련된 일체의 청구권을 포기한다.

제2조 (상호 청구권 포기)
   양 당사자는 본 사안과 관련하여 상대방에 대한 일체의 민·형사상 청구 및 진정·고소·고발을
   하지 아니한다.
   이미 진행 중인 절차가 있을 경우 본 합의서 서명 후 5일 이내 취하한다.

제3조 (비방·공개 금지)
   양 당사자는 본 사안 및 본 합의 내용을 외부에 공개하거나 상대방을 비방하지 아니한다.
   SNS·언론·인터뷰 등 어떠한 매체에서도 본 사안을 언급하지 아니한다.

제4조 (위반 시 책임)
   본 합의서 조항을 위반한 경우, 위반 당사자는 상대방에게 위약금 [____원]을 즉시 지급한다.
   본 위약금은 손해배상예정의 성격을 가지며, 실손해가 그를 초과하는 경우 추가 청구할 수 있다.

제5조 (효력)
   본 합의서는 양 당사자의 서명·날인과 동시에 효력이 발생하며,
   이후 어떠한 사유로도 무효·취소를 주장할 수 없다.

제6조 (관할)
   본 합의서에 관한 분쟁 발생 시 [서울중앙지방법원]을 제1심 관할법원으로 한다.

본 합의서를 증명하기 위하여 2부 작성하여 각자 1부씩 보관한다.

20    년    월    일

   "갑": 성명/상호: ________________  주소: ________________
         서명/날인: (인)

   "을": 성명/상호: ________________  주소: ________________
         서명/날인: (인)

   (필요 시) 입회인:
        성명: ________________  관계: ________________
        서명/날인: (인)
```

## 양식 2. 출연료·수강료 정산 합의서 (간단)

```
                       정산 합의서

[A]와 [B]는 [작품명/계약명]에 관하여 다음과 같이 정산을 합의한다.

1. 정산 사유: [____________________________]
2. 정산 금액: 일금 ____________원
3. 지급일: 20XX년 XX월 XX일
4. 지급계좌: ________ 은행 ________________________

위 금액 지급 시 본 [계약/사안]에 관한 일체의 청구권은 소멸한다.

20    년    월    일

   [A]: 성명 ____________ 서명 (인)
   [B]: 성명 ____________ 서명 (인)
```

## 양식 3. 환불 합의서 (멤버 환불 분쟁)

```
                       수강료 환불 합의서

주식회사 케이디포 액팅스튜디오(이하 "회사")와 [멤버 성명]는 [클래스명] 수강에 관하여
다음과 같이 환불을 합의한다.

1. 수강 정보
   - 클래스명: ____________
   - 결제일·금액: 20XX년 XX월 XX일 / ____________원
   - 진행 회차: ___회 / 총 ___회

2. 환불 금액: 일금 ____________원
   - 산정 근거: 학원법 시행령 별표 4 (총 수강시간의 [1/3 / 1/2 / ...] 경과 기준)
   - [또는] 회사와 멤버 간 합의에 의함

3. 환불 방법
   - 결제수단: [카드 취소 / 계좌 입금]
   - 입금 계좌: ________ 은행 ________________________
   - 처리 기한: 본 합의서 서명일로부터 5영업일 이내

4. 청구권 포기
   멤버는 본 환불금 수령으로 본 클래스에 관한 일체의 청구권 (추가 환불·손해배상 등)을
   포기한다.

5. 비공개
   양 당사자는 본 합의 내용을 외부에 공개하지 아니한다.

20    년    월    일

   회사: 주식회사 케이디포 액팅스튜디오  대표이사 권 동 원 (인)
   멤버: 성명 ____________ 서명 (인)
   (미성년) 법정대리인 ____________ 서명 (인)
```

## 양식 4. 비밀유지·경업금지 각서 (퇴직 시)

```
                       각  서

본인 [성명]은 주식회사 케이디포 액팅스튜디오(이하 "회사")에서 [직책]으로 재직하였고,
20XX년 XX월 XX일자로 퇴직함에 있어 다음을 확약합니다.

1. 비밀유지
   본인은 재직 중 알게 된 회사의 영업비밀(멤버·강사·고객 정보, 교육 콘텐츠, 교재,
   커리큘럼, 마케팅 노하우, 재무 정보 등)을 퇴직 후 [3]년간 외부에 누설하거나
   본인 또는 제3자의 이익을 위하여 사용하지 아니합니다.

2. 경업금지
   본인은 퇴직일로부터 [1]년간 다음 행위를 하지 아니합니다.
   가. 회사 사업장 반경 10km 이내에서 동종 학원 설립·운영
   나. 회사의 멤버·강사를 동종 사업으로 유인·전직 권유
   다. 회사의 거래처와 직접 거래

3. 자료 반환
   본인은 퇴직 시 회사로부터 제공받거나 업무 중 보관한 모든 자료(PDF·디지털 파일·종이 등)를
   회사에 반환하거나 파기합니다. 개인 보관본은 남기지 아니합니다.

4. 위반 시 책임
   본인이 위 약속을 위반한 경우, 회사에 위약금 [____원] 및 회사가 입은 실손해를 배상합니다.
   본 위약금은 손해배상예정의 성격을 가집니다.

5. 본 각서의 효력
   본 각서는 본인의 자유의사에 의해 작성되었으며, 어떠한 사유로도 무효·취소를 주장하지 않습니다.

20    년    월    일

   확약자:  성명: ____________ 서명 (인)
            주민등록번호 앞자리: ________
            연락처: ________________________
            주소: ________________________

   확인자(회사):
            주식회사 케이디포 액팅스튜디오
            대표이사 권 동 원 (인)
```

> ⚠️ **경업금지 약정 한계**:
> - 기간 1년·동일 권역·반대급부 보상 — 합리적 한계
> - 2년 이상·전국·무보상 — 무효 위험
> - 단순 강사는 경업금지 적용 좁게 해석됨

## 양식 5. 미성년 보호자 동의서

```
                  미성년자 활동 동의서 (보호자용)

본인은 [미성년자 성명] (생년월일 ____________)의 법정대리인으로서,
다음 활동에 대하여 동의합니다.

1. 활동 내용: [클래스 수강 / 출연 / 발표회 / 촬영 등]
2. 활동 기간: 20XX년 XX월 XX일 ~ 20XX년 XX월 XX일
3. 활동 장소: ________________________
4. 활동 시간: ________________________ (1일 ___시간 한도)
5. 안전관리: 보호자 동반 [필수 / 선택], 매니저 동행 [필수 / 선택]

본인은 다음 사항을 확인합니다.
- 본 활동의 내용·일정·위험성을 충분히 안내받았습니다.
- 응급 상황 시 회사가 본인에게 즉시 연락 후 합리적 조치를 취할 권한을 부여합니다.
- 본인의 동의는 본인의 자유의사에 의한 것입니다.

20    년    월    일

   법정대리인(부): 성명 ____________ 서명 (인)
                  연락처: ________________________
                  관계: ________________________

   법정대리인(모): 성명 ____________ 서명 (인)
                  연락처: ________________________
                  관계: ________________________

   ※ 친권의 공동행사 원칙 (민법 제909조)에 따라 부모 양측 동의 권장.
   ※ 부모 일방 단독 동의 시 무효 위험 (다른 부모가 사후 무효 주장 가능).
```

## 양식 6. 약식 정정·사과문 (SNS 비방 합의 시)

```
                       정 정 사 과 문

본인은 20XX년 XX월 XX일 [본인의 SNS 계정명·URL]에 게시한
주식회사 케이디포 액팅스튜디오 관련 게시물에 사실과 다른 내용이 포함되어 있어
다음과 같이 정정 및 사과합니다.

1. 정정 내용
   "[원게시물의 인용 부분]" → [정확한 사실]

2. 사과
   위 게시물로 인하여 [회사명]과 관련자분들께 심려를 끼쳐드린 점 진심으로 사과드립니다.
   향후 동일·유사한 사실관계 미확인 게시를 하지 않을 것을 약속드립니다.

3. 게시 조건
   본 정정 사과문은 [게시 매체]에 [3]일 이상 상단 고정 게시한 후
   원게시물과 함께 삭제됩니다.

20    년    월    일

   작성자: ________________________ (서명/인)
```

## 공증 vs 자필

| 형식 | 강제집행 | 추천 상황 |
|---|---|---|
| 자필 합의서 | 별도 소송 후 가능 | 소액·단순 분쟁 |
| 사서증서 인증 (공증) | 별도 소송 후 가능 | 중간 |
| **공정증서 (공증인 작성)** | **즉시 강제집행 가능** | 금액 큼·이행 불안 |

→ 100만원 이상 + 분할 지급 + 이행 불안 → **공정증서** 권장
→ 합의 즉시 일시불 + 단순 → 자필 합의서로 충분

## 합의서 작성 시 자주 빠뜨리는 5가지

1. **합의 사유 명시** ("본 사안" 정의 없으면 합의 범위 모호)
2. **청구권 포기 범위** (특정 분쟁만? 일체? 형사 포함?)
3. **위반 시 책임** (위약금 + 실손해 청구권)
4. **비공개·비방금지** (특히 SNS·언론)
5. **관할법원** (이행 불이행 시 절차 명확)
