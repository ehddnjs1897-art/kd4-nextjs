# KD4 인사이트 저장 크롬 확장 프로그램

웹페이지에서 **우클릭 → "KD4 인사이트에 저장"** 한 번이면 끝.
자동으로 `/api/insights`로 전송되어 Gemini AI가 카테고리·태그·3줄 요약을 생성합니다.

## 설치 방법 (한 번만)

1. 크롬 주소창에 `chrome://extensions/` 입력
2. 우상단 **"개발자 모드"** 토글 켜기
3. 좌상단 **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. 이 `chrome-extension/` 폴더 선택
5. 설치 완료 — 툴바에 골드 원 아이콘이 생김

## API URL 설정

설치 후 최초 1회:

1. 툴바의 확장 프로그램 아이콘 클릭 → 옵션 창 열림
2. API URL 입력:
   - 로컬 테스트: `http://localhost:3000`
   - 배포 버전: `https://your-domain.vercel.app`
3. **저장** 클릭

## 사용법

아무 웹페이지에서 마우스 **우클릭** → 세 가지 옵션:

| 메뉴 | 동작 |
|------|------|
| KD4 인사이트에 저장 | 현재 페이지 URL만 저장 |
| KD4 인사이트에 저장 (선택 텍스트 포함) | 드래그한 텍스트를 메모로 같이 저장 |
| KD4 인사이트에 이 링크 저장 | 특정 링크(`<a>` 태그) 우클릭 시 노출 — 그 링크 저장 |

저장 시작 → "저장 중..." 알림 → 완료되면 카테고리와 제목이 알림으로 뜸.

## 트러블슈팅

- **저장 실패 (Failed to fetch)**: 옵션에서 API URL 확인. 서버가 꺼져 있지 않은지 확인.
- **CORS 에러**: 서버 코드의 `Access-Control-Allow-Origin: *` 헤더 확인 (이미 포함됨).
- **우클릭 메뉴가 안 보임**: `chrome://extensions/`에서 확장 프로그램이 "사용 설정"인지 확인 후 새로고침.

## 파일 구조

```
chrome-extension/
  manifest.json    # 확장 설정 (Manifest V3)
  background.js    # 우클릭 메뉴 등록 + API 호출
  options.html     # API URL 설정 페이지
  options.js       # 설정 저장 로직
  icon.png         # 툴바 아이콘 (128x128)
  README.md        # 이 파일
```
